# TECHCROSS P&ID AI 검토 시스템 관련 확인 요청 사항

> 작성일: 2025-12-26

안녕하세요, TECHCROSS 기본설계팀 담당자님.

P&ID AI 검토 시스템 개발을 위해 아래 사항들에 대한 확인 및 자료 요청드립니다.

질문 항목이 다소 많아 양해 부탁드립니다. 정확한 시스템 구현을 위해 사전에 확인이 필요한 부분들을 정리하였습니다. 한 번에 회신이 어려우시면 항목별로 나누어 별도 회신 주셔도 괜찮습니다.

---

## 1-1. P&ID 인식 및 체크리스트 검토 관련

1) 체크리스트 Excel 파일(Check list_P&ID_ECS-B.xlsm, Check list_P&ID_HYCHLOR 2.0.xlsm)이 손상되어 열리지 않습니다. 정상 파일로 재전송 부탁드립니다.

2) 체크리스트 60개 항목 중 자동 검증이 가능한 항목과 수동 검토가 필요한 항목의 구분 기준이 있다면 알려주세요.

3) 판정 결과(충족/미충족/N/A) 중 N/A가 적용되는 조건이 궁금합니다. 특정 프로젝트나 선주사 요구에 따라 달라지는지요?

4) BWMS 장비 8종(ECU, FMU, ANU, TSU, APU, GDS, EWU, CPC 등) 외에 추가로 인식해야 하는 심볼이 있는지 확인 부탁드립니다.

5) P&ID 도면의 NOTE 영역에 기재된 설계 규칙(예: TSU-APU 5m 이내, ANU Injection Port 10m 이내)과 체크리스트 항목 간의 매핑 관계가 있다면 공유 부탁드립니다.

6) 검토 결과 불일치 항목 발생 시 담당자 알림이나 승인 프로세스가 있는지요?

---

## 1-2. Valve Signal List 작성 관련

7) "SIGNAL FOR BWMS" 텍스트 라벨이 붙은 밸브만 Signal List 대상인지, 아니면 다른 식별 기준이 있는지 확인 부탁드립니다.

8) 밸브 카테고리(Required valve, Alarm By-pass valve, PUMP SIGNAL, ABNORMAL/INTERLOCK)의 판별 기준이 도면상 어떻게 표시되는지 알려주세요.

9) SIGNAL 조건 텍스트(예: "V/V:LITTLE OPEN=SIGNAL:ON")는 도면에서 직접 추출해야 하는지, 아니면 별도 규칙 테이블이 존재하는지요?

10) 운전 모드(BALLASTING, DE-BALLASTING, MIXING, STRIPPING)별로 Signal List가 달라지는지, 통합 관리되는지 궁금합니다.

---

## 1-3. Equipment List 작성 관련

11) 장비 수량(Quantity) 정보가 도면에서 추출 가능한지, 아니면 별도 입력이 필요한지요?

12) P&ID 도면의 '*' 마크(MAKER SUPPLY)가 붙은 장비만 Equipment List 대상인지 확인 부탁드립니다.

13) CONNECTION SIZE 정보가 ECS COMPONENTS LIST에 있는데, 이 정보도 AI가 자동 추출해야 하는지요?

---

## 1-4. Deviation List 작성 관련

14) Deviation 항목의 판단 기준이 되는 POR(Purchase Order Requirement) 원본 문서가 있는지요?

15) "구매자 결정" 항목은 AI가 제안하고 사람이 최종 결정하는 방식인지, 아니면 기존 이력 기반으로 자동 추천하는 방식인지요?

16) 영업팀 확인이 필요한 항목의 자동 분류 기준이 있다면 알려주세요.

---

## 공통 요청 사항

17) ECS와 HYCHLOR 2.0 시스템 간 체크리스트 항목이나 설계 규칙에 차이가 있다면 정리된 문서가 있는지요?

18) 실제 운영 중인 체크리스트 검토 완료 샘플(작성된 체크리스트 예시)이 있다면 참고용으로 공유 부탁드립니다.

19) 선주사별로 요구사항이 다른 경우 커스터마이징 범위가 어느 정도인지 알려주세요.

20) DWG 원본 파일 활용이 가능한지, 아니면 PDF만으로 진행해야 하는지 확인 부탁드립니다. DWG 활용 시 심볼 인식 정확도를 높일 수 있습니다.

21) 현재 사용 중인 심볼 라이브러리(CAD 블록)가 있다면 공유 부탁드립니다.

22) AI 학습을 위해 BWMS 장비별 개별 심볼 이미지(PNG 또는 SVG)가 필요합니다. ECU, FMU, ANU, TSU, APU, GDS, EWU, CPC 등 주요 장비의 심볼 이미지를 별도 파일로 제공 가능한지요?

23) 와이어프레임에 나온 "승인도 업로드" 기능에서 지원해야 하는 파일 형식(PDF, DWG, 이미지 등)을 알려주세요.

24) 검토 결과 리포트의 선호 출력 형식(Excel, PDF, 웹 대시보드 등)이 있으신지요?

25) 기존에 검토 완료된 프로젝트 사례(Valve Signal List, Equipment List, Deviation List 작성 완료본)가 있다면 학습 데이터로 활용 가능한지 확인 부탁드립니다.
