TECHCROSS P&ID AI 검토 시스템 관련 확인 요청 사항

작성일: 2025-12-27

안녕하세요, TECHCROSS 기본설계팀 담당자님.

P&ID AI 검토 시스템 개발을 위해 아래 사항들에 대한 확인 및 자료 요청드립니다.
정확한 시스템 구현을 위해 사전에 확인이 필요한 부분들을 정리하였습니다.
한 번에 회신이 어려우시면 항목별로 나누어 별도 회신 주셔도 괜찮습니다.

================================================================================

[공통] 자료 요청 및 개발 범위 확인

1. 전달받은 체크리스트 xlsm 파일이 모두 손상되어 열리지 않습니다.
   아래 3개 파일의 정상 버전 재전송 부탁드립니다.
   - 하이클로 예시: (Ver.25.01) Check list_P&ID_HYCHLOR 2.0 (2025.10.01).xlsm
   - 와이어프레임: (Ver.25.01) Check list_P&ID_ECS-B (2025.06.23)_김수민.xlsm
   - ECS 예시: (Ver.25.01) Check list_P&ID_ECS-B (2025.12.19.).xlsm

2. ECS와 HYCHLOR 2.0 시스템 간 체크리스트 항목이나 설계 규칙에 차이가 있다면
   정리된 문서가 있는지요?

3. AI가 도면에서 장비(ECU, FMU, ANU 등)를 자동 인식하려면 참조 자료가 필요합니다.
   아래 중 제공 가능한 자료가 있다면 공유 부탁드립니다.
   - DWG 원본 도면 (PDF 대비 인식 정확도 향상)
   - 표준 심볼 라이브러리 (AutoCAD 블록 파일 등)
   - 장비별 심볼 이미지 (PNG, JPG 등)

4. 기존 검토 완료 사례가 있다면 AI 개발 및 테스트용 참고 자료로 공유 부탁드립니다.
   - 작성 완료된 체크리스트 샘플
   - Valve Signal List / Equipment List / Deviation List 완료본

5. 검토 결과 리포트의 선호 출력 형식(Excel, PDF, 웹 대시보드 등)이 있으신지요?

6. 선주사별로 요구사항이 다른 경우 커스터마이징 범위가 어느 정도인지 알려주세요.

================================================================================

[1-1. P&ID 인식 및 체크리스트 검토]

1. 체크리스트 60개 항목의 판정 관련 기준이 궁금합니다.
   - AI가 도면만 보고 자동 판정 가능한 항목 vs 수동 검토 필요 항목의 구분 기준
   - N/A가 적용되는 조건 (프로젝트나 선주사 요구에 따라 달라지는지)

2. BWMS 장비(ECU, FMU, ANU, TSU, APU, GDS, EWU, CPC, FTS, CSU, PDE, PCU, FCV 등) 중
   P&ID에서 AI가 인식해야 하는 장비의 전체 목록을 확인 부탁드립니다.

3. 체크리스트 60개 항목 판정 시 참조하는 설계 규칙(장비 간 이격 거리, 배치 순서 등)이
   어디에 문서화되어 있는지요? (P&ID 도면 내, 별도 설계 기준서 등)
   체크리스트 항목과의 대응 관계가 있다면 공유 부탁드립니다.

================================================================================

[1-2. Valve Signal List 작성]

1. Signal List 대상 밸브의 식별 및 분류 기준이 궁금합니다.
   - "SIGNAL FOR BWMS" 라벨이 붙은 밸브만 대상인지, 다른 기준이 있는지
   - 밸브 카테고리(Required valve, Alarm By-pass 등) 판별 기준이 도면상 어떻게 표시되는지

2. SIGNAL 조건 텍스트(예: "V/V:LITTLE OPEN=SIGNAL:ON")는 도면에서 직접 추출해야 하는지,
   아니면 별도 규칙 테이블이 존재하는지요?

3. 운전 모드(BALLASTING, DE-BALLASTING, MIXING, STRIPPING)별로 Signal List가 달라지는지,
   통합 관리되는지 궁금합니다.

================================================================================

[1-3. Equipment List 작성]

1. Equipment List 작성 시 정보 추출 기준이 궁금합니다.
   - '*' 마크(MAKER SUPPLY)가 붙은 장비만 대상인지
   - Quantity 정보가 도면에서 추출 가능한지, 별도 입력이 필요한지
   - CONNECTION SIZE 정보도 AI가 자동 추출해야 하는지

================================================================================

[1-4. Deviation List 작성]

1. Deviation 항목의 판단 기준이 되는 POR(Purchase Order Requirement) 원본 문서가 있는지요?

2. "구매자 결정" 항목은 AI가 제안하고 사람이 최종 결정하는 방식인지,
   아니면 과거 유사 프로젝트 사례를 참고하여 자동 추천하는 방식인지요?

3. 영업팀 확인이 필요한 항목의 자동 분류 기준이 있다면 알려주세요.

================================================================================

[기타]

1. 와이어프레임에 나온 "승인도 업로드" 기능에서 지원해야 하는 파일 형식(PDF, DWG, 이미지 등)을 알려주세요.
