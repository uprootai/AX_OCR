  │                                                                  │

1. Equipment List 정보 추출 기준
--------------------------------------------------------------------------------

[질문 내용]
Equipment List 작성 시 정보 추출 기준이 궁금합니다.
- '*' 마크(MAKER SUPPLY)가 붙은 장비만 대상인지
- Quantity 정보가 도면에서 추출 가능한지, 별도 입력이 필요한지
- CONNECTION SIZE 정보도 AI가 자동 추출해야 하는지

[왜 필요한가?]

  Equipment List에는 여러 정보가 포함되어야 함

  ┌──────────────────────────────────────────────────────────────────┐
  │  Equipment List 필수 정보                                        │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │  1. 추출 대상 장비 (어떤 장비를 뽑을 것인가?)                     │
  │  2. 수량 (Quantity)                                              │
  │  3. 사이즈/스펙 (CONNECTION SIZE 등)                              │
  │                                                                  │
  └──────────────────────────────────────────────────────────────────┘

[1. 추출 대상 장비 ('*' 마크 기준)]

  P&ID에는 여러 공급처의 장비가 혼재

  ┌──────────────────────────────────────────────────────────────────┐
  │  P&ID 도면 범례                                                  │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │  * TECHCROSS PROVIDES ONLY '*' MARKED EQUIPMENT                  │
  │  ○ COUNTER FLANGE WITH GASKET (SUPPLIED BY MAKER)               │
  │                                                                  │
  └──────────────────────────────────────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────┐
  │  도면 내 장비 표시                                               │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │  ┌─────────┐         ┌─────────┐         ┌─────────┐           │
  │  │   ECU   │ *       │  PUMP   │         │  VALVE  │ ○        │
  │  │         │         │         │         │         │           │
  │  └─────────┘         └─────────┘         └─────────┘           │
  │       │                   │                   │                │
  │       ↓                   ↓                   ↓                │
  │  TECHCROSS 공급      조선소 공급        TECHCROSS 공급         │
  │  (추출 대상)         (제외)            (Counter Flange)        │
  │                                                                  │
  └──────────────────────────────────────────────────────────────────┘

  확인 필요:
  • '*' 마크만 추출?
  • '○' 마크도 추출? (Counter Flange 등 부자재)

[2. 수량(Quantity) 추출 방법]

  ┌──────────────────────────────────────────────────────────────────┐
  │  수량 정보 획득 방법                                             │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │  방법 1: 도면에서 자동 카운트                                    │
  │  ┌────────────────────────────────────────────────────────────┐ │
  │  │                                                            │ │
  │  │  P&ID 도면에서 ECU 심볼 검출                               │ │
  │  │                                                            │ │
  │  │  ┌───┐       ┌───┐                                        │ │
  │  │  │ECU│       │ECU│     → ECU 2개 검출                     │ │
  │  │  └───┘       └───┘                                        │ │
  │  │                                                            │ │
  │  │  → AI가 자동으로 개수 파악                                 │ │
  │  │                                                            │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                  │
  │  방법 2: 도면 내 텍스트로 표시                                   │
  │  ┌────────────────────────────────────────────────────────────┐ │
  │  │                                                            │ │
  │  │  장비 옆에 "QTY: 2" 또는 "x2" 텍스트                       │ │
  │  │                                                            │ │
  │  │  ┌───┐                                                     │ │
  │  │  │ECU│ (x2)  → OCR로 추출                                 │ │
  │  │  └───┘                                                     │ │
  │  │                                                            │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                  │
  │  방법 3: 별도 입력 필요                                          │
  │  ┌────────────────────────────────────────────────────────────┐ │
  │  │                                                            │ │
  │  │  도면에 수량 정보 없음                                     │ │
  │  │  → 사용자가 수동 입력                                      │ │
  │  │  → 또는 BOM 문서에서 참조                                  │ │
  │  │                                                            │ │
  │  └────────────────────────────────────────────────────────────┘ │
  │                                                                  │
  └──────────────────────────────────────────────────────────────────┘

[3. CONNECTION SIZE 정보 출처]

  Equipment List에는 사이즈/스펙 정보가 포함

  ┌──────────────────────────────────────────────────────────────────┐
  │  ECS COMPONENTS LIST 예시                                        │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │  장비       │ MODEL    │ CONNECTION SIZE                        │
  │  ───────────┼──────────┼────────────────────────────────        │
  │  ECU        │ ECU1000B │ IN: 10K-250A, OUT: 10K-250A            │
  │  FMU        │ 5K-500A  │ IN: 10K-250A, OUT: 10K-250A            │
  │  T-STRAINER │ 5K-450A  │ IN: 10K-250A, OUT: 10K-250A            │
  │                                                                  │
  └──────────────────────────────────────────────────────────────────┘

  사이즈 정보 출처 가능성:

  ┌────────────────────────────────────────────────────────────────┐
  │  출처 1: P&ID 도면에서 추출                                     │
  │                                                                │
  │  도면 내 장비 옆에 사이즈 표시                                 │
  │                                                                │
  │  ┌─────────┐                                                   │
  │  │   FMU   │ 5K-500A  ← OCR로 추출                            │
  │  └─────────┘                                                   │
  └────────────────────────────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────────┐
  │  출처 2: 배관 라인 번호에서 추출                                │
  │                                                                │
  │  10K-250A-BW001  ← 라인 번호에 사이즈 포함                    │
  │       │                                                        │
  │       └─ 10K-250A = 10K 플랜지, 250A 사이즈                   │
  └────────────────────────────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────────┐
  │  출처 3: 별도 문서 참조                                         │
  │                                                                │
  │  ECS COMPONENTS LIST 문서에서 참조                            │
  │  → P&ID 분석과 별개로 문서 파싱 필요                          │
  └────────────────────────────────────────────────────────────────┘

[개발 영향]

  ┌────────────────────┬────────────────────────────────────────────┐
  │  정보 출처          │  개발 방식                                 │
  ├────────────────────┼────────────────────────────────────────────┤
  │  도면에서 추출      │  YOLO + OCR 활용                          │
  │  별도 문서 참조     │  추가 문서 파싱 로직 필요                  │
  │  수동 입력          │  입력 UI 필요                              │
  └────────────────────┴────────────────────────────────────────────┘

[핵심]
  → 추출 대상 범위, 수량 획득 방법, 사이즈 정보 출처 확인 필요
