## 1. 메일 목적

이번 메일의 목적은 아래 3가지다.

1. 2차 미팅에서 새로 정리된 방향을 고객과 다시 맞춘다.
2. 3차 미팅에서 반드시 결론 내야 하는 항목을 사전에 공유한다.
3. 가능하면 미팅 전에 자료를 먼저 받아 미팅 시간을 줄인다.

---

## 2. 메일에 반드시 들어갈 항목

### 2.1 2차 미팅 이후 내부 정리 내용

- 현재 1차 목표는 `모든 숫자 추출`이 아니라 `내경 / 외경 / 폭` 우선 추출
- 대상은 `하얀색 단품 도면`
- `사양서`, `품질기준서`는 현재 1차 자동화 범위에서 제외
- `구멍 수`, `가공 형태` 등은 2단계 검토 항목
- 도면별로 `내경 / 외경 / 폭` 정답 위치는 전문가 판단이 필요한 경우가 있음

### 2.2 고객 확인이 필요한 미결 항목

- `SS275` 스펙 자료 전달 가능 여부
- 실제 견적서 양식 공유 가능 여부
- `내경 / 외경 / 폭` 이후 2단계로 들어갈 항목의 우선순위
- 구멍 수, 가공 형태, 홀 표기 해석 중 무엇이 먼저 필요한지
- 추출값이 견적서 어느 칸과 직접 연결되는지
- 대표 도면 몇 개에 대해 `내경 / 외경 / 폭` 위치를 표시한 샘플 공유 가능 여부
- `DWG` 보유 여부 및 제공 가능 여부

### 2.3 3차 미팅에서 확정해야 할 항목

- 자동 추출 1순위 항목을 `내경 / 외경 / 폭`으로 확정
- 2단계 항목 범위 확정
- 견적 산출 로직의 입력값 매핑 확정
- 고객이 실제로 원하는 결과물 형식 확정

## 3. 3차 미팅 사전 질문 목록

아래 질문은 메일 본문에 그대로 넣거나 체크리스트 형태로 정리해도 된다.

| 구분 | 질문 | 목적 |
|------|------|------|
| 소재 | `SS275` 관련 원본 스펙 자료를 전달해주실 수 있는지 | 재질 판정 기준 확보 |
| 견적 | 현재 사용 중인 견적서 양식을 공유 가능한지 | 출력 포맷 설계 |
| 치수 | 현재 1차로 필요한 값이 `내경`, `외경`, `폭` 3개가 맞는지 다시 확인 부탁드리는지 | 범위 고정 |
| 샘플 | 대표 도면 몇 개에 대해 `내경`, `외경`, `폭` 위치를 표시한 샘플을 공유 가능한지 | 정답 기준 확보 |
| 가공 | `내경 / 외경 / 폭` 다음 단계에서 우선 봐야 할 항목이 `구멍 수`, `가공 형태`, `홀 표기` 중 무엇인지 | 2단계 우선순위 확정 |
| 견적 연결 | 추출한 `내경 / 외경 / 폭`이 견적서상 어느 항목 판단에 직접 쓰이는지 | 파이프라인 연결 |
| CAD | 원본 `DWG` 보유 자료가 있는지, 제공 가능한지 | 해석 가능성 검토 |

### 3.1 내부 참고용 별도 질문

아래 항목은 고객 메일에 바로 넣기보다 내부 검토 후 필요 시 별도 질문으로 분리하는 편이 낫다.

- 외주 견적 요청서 양식 추가 요청 여부
- 협력사 단가 산정 방식 상세 요청 여부

## 6. 첨부 또는 링크로 같이 보내면 좋은 자료

- `내경 / 외경 / 폭` 추출 예시 1~2장
- 전문가가 정답 위치를 표시한 샘플 도면 몇 장
- 현재 자동화 범위 / 다음 단계 범위 1페이지 정리본
- 필요한 경우 견적서 연결 가정안 1장

## 7. 작성 우선순위

메일 작성 시 우선순위는 아래 순서가 적절하다.

1. 고객이 답해야 하는 질문을 먼저 정리
2. 미팅 전에 받아야 하는 자료를 분리
3. 3차 미팅에서 확정할 의제를 3개 이하로 압축
4. 확인되지 않은 가정은 메일 본문에 단정적으로 쓰지 않기
