## 조립 순서

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: FIXED WEDGE 설치                                │
│          - 베어링 케이싱 측면에 FIXED WEDGE (②) 위치 결정│
│          - 하우징 볼트로 고정                            │
│                                                         │
│  Step 2: MOVING WEDGE 조립                               │
│          - FIXED WEDGE 위에 MOVING WEDGE (①) 배치       │
│          - 경사면이 서로 맞물리도록 정렬                  │
│                                                         │
│  Step 3: LOCKING PIN 체결                                │
│          - NORD LOCK WASHER (⑤) 장착 (3개)             │
│          - LOCKING PIN (③) 삽입 (3개)                   │
│          - HEX NUT (④)로 체결                           │
│                                                         │
│  Step 4: 측면 위치 조정                                  │
│          - LOCKING PIN을 조여 MOVING WEDGE 이동          │
│          - 경사면 작용으로 측면 위치 조정                │
│          - 목표 위치 달성 후 최종 체결                   │
└─────────────────────────────────────────────────────────┘
```

## 기능 설명

### 측면 조정 웨지 조립체의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 베어링 측면 위치 조정 (Lateral Position Adjustment)  │
│     - MOVING WEDGE 이동 → 측면 위치 변화                │
│     - 웨지 경사 각도에 따른 정밀 조정                    │
│                                                         │
│  2. 베어링 중심 정렬 (Centerline Alignment)              │
│     - 저널 베어링의 측면 정렬                           │
│     - 샤프트 중심과 베어링 중심 일치                     │
│                                                         │
│  3. 열팽창 보상 (Thermal Expansion Compensation)         │
│     - 운전 중 열팽창에 의한 위치 변화 보상               │
│     - 조정 후 LOCKING PIN으로 위치 고정                  │
└─────────────────────────────────────────────────────────┘
```
