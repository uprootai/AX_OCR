## 기능 설명

### 피벗의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 틸팅 패드 지지점 (Tilting Pad Support)               │
│     - 저널 베어링의 틸팅 패드 지지                       │
│     - 볼 끝단이 패드와 점 접촉                           │
│                                                         │
│  2. 하중 전달 (Load Transfer)                            │
│     - 샤프트 하중을 패드에 전달                          │
│     - 점 접촉으로 자유로운 틸팅 허용                     │
│                                                         │
│  3. 자동 정렬 (Self-Alignment)                           │
│     - 축 편심, 미스얼라인먼트 보상                       │
│     - 유막 형성 최적화                                   │
└─────────────────────────────────────────────────────────┘
```

### 틸팅 패드 베어링 구조

```
           샤프트 (Shaft)
              ↓
    ┌─────────────────────┐
    │   ◯◯◯◯◯◯◯◯◯◯◯◯◯     │  ← 오일 필름
    ├─────────────────────┤
    │    Tilting Pad      │  ← 틸팅 패드
    └──────────┬──────────┘
               │
           ┌───┴───┐
           │ Ball  │  ← 피벗 (PIVOT)
           │  ●    │     PT.1~PT.9
           └───────┘
               │
    ───────────┴───────────  ← 베어링 케이싱
```
