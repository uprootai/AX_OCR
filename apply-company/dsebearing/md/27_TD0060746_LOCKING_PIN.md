## 기능 설명

### 로킹 핀의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 웨지 고정 (Wedge Locking)                            │
│     - MOVING WEDGE/FIXED WEDGE 위치 고정                │
│     - M16×1.5 미세 피치로 정밀 체결                      │
│                                                         │
│  2. 풀림 방지 (Anti-Loosening)                           │
│     - Ø14 생크부가 웨지 홀에 삽입                        │
│     - 3mm 홈에 와이어/록킹 와셔 장착                     │
│                                                         │
│  3. 고온 환경 대응 (High Temperature)                    │
│     - ASTM A193 B16 고온용 합금강                       │
│     - 538°C까지 안정적 체결력 유지                       │
└─────────────────────────────────────────────────────────┘
```
