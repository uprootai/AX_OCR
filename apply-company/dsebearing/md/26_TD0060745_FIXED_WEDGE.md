## NOTES (주의사항)

### 한국어
1. **명시되지 않은 FILLET치수**는 TOOL규율을 따를 것 (단 1MM이하)
2. **명시되지 않은 TAP DRILL치수**는 가상의 나사산 2개를 돌리는 깊이로 할 것
3. **모든 나사 TAP HOLES**는 **GO, NO GO-GAUGE**로 검사할 것
4. **명시되지 않은 치수**는 CENTER를 기준으로 대칭임
5. **날카로운 모서리부, BURR**는 제거할 것
6. **볼트구멍 및 PIN HOLE등**에 들어 있는 이물질을 모두 제거할 것

### English
1. FOLLOW TYPICAL **TOOL RADIUS** NOT SHOWN FILLET DIM. (ONLY BELOW 1MM)
2. TAP DRILL DEPTH WILL BE EXTENDED INCLUDING **VIRTUAL THREADS 2EA** IN CASE OF NOT SHOWN
3. CHECK ALL **TAP HOLES** WITH **GO, NO GO-GAUGE**
4. FOLLOW **SYMMETRY FOR CENTER LINE** FOR NOT SHOWN DIMENSION
5. **REMOVE SHARP EDGE AND BURR**
6. **REMOVE FOREIGN MATERIAL** IN BOLT HOLE AND PIN HOLE ETC.

## 기능 설명

### 고정 웨지의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 베어링 위치 고정 (Position Fixing)                   │
│     - 무빙 웨지(MOVING WEDGE)의 기준면 제공             │
│     - 경사면이 무빙 웨지와 맞물림                        │
│                                                         │
│  2. 하중 지지 (Load Support)                             │
│     - 베어링 하중을 하우징에 전달                        │
│     - SM490A 구조용강으로 높은 강성                      │
│                                                         │
│  3. 수직력 전환 (Vertical Force Conversion)              │
│     - 무빙 웨지 수평 이동 → 수직 위치 변환 기준면        │
│     - 23° 경사 각도로 정밀 변환비 제공                   │
└─────────────────────────────────────────────────────────┘
```

### MOVING WEDGE vs FIXED WEDGE 비교

| 항목 | TD0060744 (MOVING WEDGE) | TD0060745 (FIXED WEDGE) |
|------|--------------------------|-------------------------|
| **재질** | S45C+N (탄소강) | SM490A (구조용강) |
| **볼트** | M16×1.5 (조정용) | M18×1.5 (고정용) |
| **경사면 위치** | 하부 | 상부 |
| **역할** | 이동 (조정) | 고정 (기준) |
| **경사 방향** | ╲ (하향) | ╱ (상향) |

### 웨지 조립 구조

```
           베어링
    ┌─────────────────┐
    │                 │
    │    BEARING      │
    │                 │
    └─────────────────┘
              │
    ┌─────────▼─────────┐
    │    MOVING WEDGE   │ ← TD0060744 (조정)
    │       ╲           │    M16×1.5 볼트로 수평 이동
    │        ╲          │
    │         ╲         │
    └──────────╲────────┘
                ╲
    ┌───────────╲───────┐
    │            ╱      │ ← TD0060745 (고정)
    │           ╱       │    M18×1.5 볼트로 하우징에 고정
    │    FIXED ╱        │
    │    WEDGE          │
    └───────────────────┘
              │
    ══════════╧═════════  ← 하우징 (HOUSING)
```

### 조정 원리

```
    볼트 조임 → 무빙 웨지 수평 이동 → 경사면 접촉 → 수직 위치 변화

    조정량 계산:
    - 경사 각도: 23°
    - 수평 이동 1mm → 수직 변화 ≈ 0.42mm (tan 23°)
    - 미세 조정 가능
```
