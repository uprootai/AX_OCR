## 조립 순서

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: FIXED WEDGE 설치                                │
│          - 베어링 하우징에 FIXED WEDGE (②) 위치 결정     │
│          - 하우징 볼트로 고정                            │
│                                                         │
│  Step 2: MOVING WEDGE 조립                               │
│          - FIXED WEDGE 위에 MOVING WEDGE (①) 배치       │
│          - 경사면이 서로 맞물리도록 정렬                  │
│                                                         │
│  Step 3: LOCKING PIN 체결                                │
│          - NORD LOCK WASHER (⑤) 장착 (3개)             │
│          - LOCKING PIN (③) 삽입 (3개)                   │
│          - HEX NUT (④)로 체결                           │
│                                                         │
│  Step 4: 위치 조정                                       │
│          - LOCKING PIN을 조여 MOVING WEDGE 수평 이동     │
│          - 경사면 작용으로 수직 위치 조정                │
│          - 목표 간극 달성 후 최종 체결                   │
└─────────────────────────────────────────────────────────┘
```

## 기능 설명

### 하부 조정 웨지 조립체의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 베어링 수직 위치 조정 (Vertical Position Adjustment) │
│     - MOVING WEDGE 수평 이동 → 수직 위치 변화           │
│     - 웨지 경사 각도에 따른 정밀 조정                    │
│                                                         │
│  2. 베어링 간극 조정 (Clearance Adjustment)              │
│     - 저널 베어링의 하부 간극 미세 조정                  │
│     - 조정 후 LOCKING PIN으로 위치 고정                  │
│                                                         │
│  3. 하중 지지 (Load Support)                             │
│     - 베어링 하중을 하우징에 전달                        │
│     - FIXED WEDGE가 주요 하중 지지                       │
└─────────────────────────────────────────────────────────┘
```
