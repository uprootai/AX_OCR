## NOTES (주의사항)

### 한국어
1. **명시되지 않은 FILLET치수**는 TOOL규율을 따를 것 (단 1MM이하)
2. **명시되지 않은 TAP DRILL치수**는 가상의 나사산 2개를 돌리는 깊이로 할 것
3. **모든 나사 TAP HOLES**는 **GO, NO GO-GAUGE**로 검사할 것
4. **명시되지 않은 치수**는 CENTER를 기준으로 대칭임
5. **날카로운 모서리부, BURR**는 제거할 것
6. **볼트구멍 및 PIN HOLE등**에 들어 있는 이물질을 모두 제거할 것
7. **CHAMFER C3**는 홀 가공부에만 적용
8. **PT.1**은 NO HOLE TYPE이며, 그 외는 소수점 1자리 포함 3자리 숫자로 홀 직경을 표시 (예: P999 = Ø99.9MM)

### English
1. FOLLOW TYPICAL **TOOL RADIUS** NOT SHOWN FILLET DIM. (ONLY BELOW 1MM)
2. TAP DRILL DEPTH WILL BE EXTENDED INCLUDING **VIRTUAL THREADS 2EA** IN CASE OF NOT SHOWN
3. CHECK ALL **TAP HOLES** WITH **GO, NO GO-GAUGE**
4. FOLLOW **SYMMETRY FOR CENTER LINE** FOR NOT SHOWN DIMENSION
5. **REMOVE SHARP EDGE AND BURR**
6. **REMOVE FOREIGN MATERIAL** IN BOLT HOLE AND PIN HOLE ETC.
7. **CHAMFER C3 ONLY FOR HOLE MACHINING**
8. **PT.1 MEAN NO HOLE TYPE** AND OTHERS CONSISTS OF 3 DIGITS INCLUDING 1 DECIMAL POINT (EX. P999 MEANS HOLE DIA 99.9MM)

## 기능 설명

### 솔리드 심 플레이트의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 대하중 지지 (Heavy Load Support)                     │
│     - 일반 심 플레이트보다 두꺼운 구조                   │
│     - SM490A 구조용강으로 높은 강성 확보                 │
│                                                         │
│  2. 간극 조정 (Clearance Adjustment)                     │
│     - 베어링과 하우징 사이 대간극 조정                   │
│     - 17.2mm ±0.05 정밀 두께 관리                       │
│                                                         │
│  3. 위치 조정 (Position Adjustment)                      │
│     - 스러스트 베어링 축방향 위치 설정                   │
│     - 4개 볼트로 고정                                    │
└─────────────────────────────────────────────────────────┘
```

### TD0060742 vs TD0060743 비교

| 항목 | TD0060742 (SHIM PLATE) | TD0060743 (SOLID SHIM PLATE) |
|------|------------------------|------------------------------|
| **재질** | SUS304 (스테인리스) | SM490A (구조용강) |
| **두께** | 0.3T, 0.5T, 1T | 17.2mm (솔리드) |
| **용도** | 미세 간극 조정 | 대간극/구조 지지 |
| **형태** | 얇은 판 | 두꺼운 블록 |
| **평면도** | 일반 | 0.01mm (정밀) |
