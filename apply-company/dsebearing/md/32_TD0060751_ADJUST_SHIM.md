## NOTES (주의사항)

### 한국어
1. **명시되지 않은 FILLET치수**는 TOOL규율을 따를 것 (단 1MM이하)
2. **명시되지 않은 TAP DRILL치수**는 가상의 나사산 2개를 돌리는 깊이로 할 것
3. **모든 나사 TAP HOLES**는 **GO, NO GO-GAUGE**로 검사할 것
4. **명시되지 않은 치수**는 CENTER를 기준으로 대칭임
5. **날카로운 모서리부, BURR**는 제거할 것
6. **볼트구멍 및 PIN HOLE등**에 들어 있는 이물질을 모두 제거할 것
7. **SHIM은 총 6장**으로 **3.0T, 0.3T, 0.2T, 0.1T, 0.08T, 0.05T** 임

### English
1. FOLLOW TYPICAL **TOOL RADIUS** NOT SHOWN FILLET DIM. (ONLY BELOW 1MM)
2. TAP DRILL DEPTH WILL BE EXTENDED INCLUDING **VIRTUAL THREADS 2EA** IN CASE OF NOT SHOWN
3. CHECK ALL **TAP HOLES** WITH **GO, NO GO-GAUGE**
4. FOLLOW **SYMMETRY FOR CENTER LINE** FOR NOT SHOWN DIMENSION
5. **REMOVE SHARP EDGE AND BURR**
6. **REMOVE FOREIGN MATERIAL** IN BOLT HOLE AND PIN HOLE ETC.
7. **SHIM IS TOTAL 6 SHEETS** WITH 3.0T, 0.3T, 0.2T, 0.1T, 0.08T, 0.05T

### 특별 주의사항
- **조립후 조정 가공**: 현장에서 조립 후 최종 가공 수행
- **3.0T 포함**: 대폭 조정이 필요한 경우 사용

## 기능 설명

### 조정 심의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 대폭 베어링 간극 조정 (Large Clearance Adjustment)   │
│     - 3.0T 두께로 최대 3mm 조정 가능                     │
│     - 6매 조합으로 최대 3.73mm 조정                      │
│                                                         │
│  2. 현장 조립 후 가공 (Site Assembly Machining)          │
│     - 조립후 조정 가공 영역 지정                         │
│     - 현장 상황에 맞춘 미세 조정                         │
│                                                         │
│  3. SIDE ADJUST WEDGE ASSY 적용                          │
│     - 측면 웨지 조립체에 사용                            │
│     - PT.1: 경사형, PT.2: 평판형                        │
└─────────────────────────────────────────────────────────┘
```
