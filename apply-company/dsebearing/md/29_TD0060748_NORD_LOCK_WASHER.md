## NOTES (주의사항)

### 한국어
1. **NORD LOCK WASHER 정품을 사용할 것**

### English
1. **USE GENUINE NORD LOCK WASHER**

> **중요**: 반드시 NORD-LOCK 정품을 사용해야 함
> - 유사품은 웨지 각도, 표면 처리가 다를 수 있음
> - 정품만 풀림 방지 성능 보장

## 기능 설명

### NORD LOCK 와셔의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 볼트 풀림 방지 (Bolt Loosening Prevention)           │
│     - 웨지 락 원리 적용                                  │
│     - 진동 환경에서도 풀림 방지                          │
│                                                         │
│  2. 축력 유지 (Clamping Force Maintenance)               │
│     - 풀림 시도 시 축력 증가                             │
│     - 체결 후 안정적 축력 유지                           │
│                                                         │
│  3. 재사용 가능 (Reusability)                            │
│     - 일반 스프링 와셔와 달리 재사용 가능                │
│     - 분해/재조립 시 동일 성능 유지                      │
└─────────────────────────────────────────────────────────┘
```

### 웨지 락 원리

```
    정상 체결 상태:               풀림 시도 시:

    ┌─────────────────┐           ┌─────────────────┐
    │   BOLT HEAD     │           │   BOLT HEAD     │
    └────────┬────────┘           └────────┬────────┘
             │                             │ ↻ 회전 시도
    ╔════════╪════════╗           ╔════════╪════════╗
    ║ ╲╱╲╱╲╱│╲╱╲╱╲╱  ║           ║ ↗ ╱╲╱╲│╲╱╲╱ ↗ ║
    ╠════════╪════════╣   →→→    ╠═══↑════╪════↑═══╣
    ║ ╱╲╱╲╱╲│╱╲╱╲╱╲  ║           ║ ↑ ╱╲╱╲│╱╲╱╲ ↑ ║
    ╚════════╪════════╝           ╚════════╪════════╝
             │                         ↑   │   ↑
                                   축력 증가 (풀림 방지)

    웨지 각도 > 나사 리드각 → 회전 시 축방향 이동 발생 → 축력 증가
```

### 스프링 와셔 vs NORD LOCK 비교

| 항목 | 스프링 와셔 | NORD LOCK 와셔 |
|------|-----------|----------------|
| **풀림 방지 원리** | 스프링력 | 웨지 락 |
| **진동 환경** | 취약 | 우수 |
| **재사용** | 불가 | 가능 |
| **축력 유지** | 감소 | 증가 |
| **비용** | 저렴 | 고가 |
| **고온 성능** | 저하 | 유지 |
