## NOTES (주의사항)

### 한국어
1. **명시되지 않은 FILLET치수**는 TOOL규율을 따를 것 (단 1MM이하)
2. **모든 나사 TAP HOLES**는 **GO, NO GO-GAUGE**로 검사할 것
3. **명시되지 않은 치수**는 CENTER를 기준으로 대칭임
4. **날카로운 모서리부, BURR**는 제거할 것
5. **볼트구멍 및 PIN HOLE등**에 들어 있는 이물질을 모두 제거할 것

### English
1. FOLLOW TYPICAL **TOOL RADIUS** NOT SHOWN FILLET DIM. (ONLY BELOW 1MM)
2. CHECK ALL **TAP HOLES** WITH **GO, NO GO-GAUGE**
3. FOLLOW **SYMMETRY FOR CENTER LINE** FOR NOT SHOWN DIMENSION
4. **REMOVE SHARP EDGE AND BURR**
5. **REMOVE FOREIGN MATERIAL** IN BOLT HOLE AND PIN HOLE ETC.

### 특별 주의사항
- **PT.101~PT.108**: "FOLLOW PR NOTE" - 프로젝트 요구사항 따를 것
- **12 (SITE SCOPE)**: 현장 가공 범위

## 기능 설명

### 심 플레이트의 역할
```
┌─────────────────────────────────────────────────────────┐
│  1. 베어링 간극 조정 (Clearance Adjustment)              │
│     - 반원형 형상으로 베어링 하우징에 삽입               │
│     - 두께 조합으로 간극 미세 조정                       │
│                                                         │
│  2. 위치 고정 (Position Fixing)                          │
│     - Ø18 볼트 홀: 고정 볼트 체결                        │
│     - Ø11 핀 홀: 위치 정렬 핀                           │
│                                                         │
│  3. 현장 조정 (Site Adjustment)                          │
│     - 12mm (SITE SCOPE) 영역 현장 가공 가능             │
│     - 설치 시 미세 조정                                  │
└─────────────────────────────────────────────────────────┘
```
