## CTQ (Critical To Quality) 개요

### CTQ란?
```
┌─────────────────────────────────────────────────────────┐
│  CTQ (Critical To Quality) = 품질 핵심 치수               │
│                                                         │
│  - 제품 기능과 성능에 직접적 영향을 미치는 치수             │
│  - 엄격한 공차 관리 필요                                  │
│  - 검사 필수 항목                                        │
│  - 불합격 시 재작업 또는 폐기                             │
└─────────────────────────────────────────────────────────┘
```
