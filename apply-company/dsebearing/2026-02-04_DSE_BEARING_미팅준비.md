## 2. 고객 Pain Point

현재 견적 프로세스가 **12일** 소요됨

| 단계 | 소요 | 내용 |
|------|------|------|
| 사양 확인 | 4일 | 도면별 재질/치수 검토 → **여기가 자동화 대상** |
| 외주 견적 | 6일 | 협력사에 견적 요청 |
| 자체 검토 | 2일 | 최종 검토 및 조정 |

고객 원문: "사양 확인하고 도면에 큰 치수에 여유치를 붙여서 일단 소재 견적을 받아야 되는 부분"

## 3. 1차 미팅 결과 (2026-02-04)

### 3.1 해결된 질문

| 질문 | 답변 | 상태 |
|------|------|------|
| SS400 vs SS275 불일치 | SS275가 정확한 스펙. SS400은 유사재질이라 대략 적어놓은 것 | ✅ 해결 |
| 사양서/품질기준서 필요 여부 | **견적에 안 씀.** BOM에 재질 다 있으니 그걸로 충분 | ✅ 해결 |
| 도면에서 뭘 뽑아야 하는지 | 큰 소재 사이즈가 1순위 (아래 상세) | ✅ 해결 |

### 3.2 BOM 계층 구조 (핵심 발견)

BOM에 색깔 구분이 있음. 이게 견적 대상 판별 기준임.

| 색깔 | 의미 | 견적 대상 |
|------|------|----------|
| **분홍색** | 최종 조립도 (Assembly) | X - 치수 추출 불필요 |
| **파란색** | 중간 조립품 (Sub-assembly) | X - 치수 추출 불필요 |
| **하얀색** | 단품 도면 | **O - 이것만 견적 대상** |

예시:
```
BOM 2번: T1 BEARING ASSY (분홍) ← 견적 안 뽑음
├── 3번: BEARING RING ASSY (파란) ← 견적 안 뽑음
│   ├── 4~17번: 개별 부품 (하양) ← 각각 견적 뽑음
│   └── 18번: 하위 조립품 (파란) ← 견적 안 뽑음
│       └── 19~23번: 부품 (하양) ← 각각 견적 뽑음
└── 단품 견적 전부 합산 = 2번 전체 견적
```

김 대표 원문: "하얀색으로 표기돼 있는 도면들을 개개별로 소재 치수부터 시작해서 단품의 견적이 다 나와야 합산 견적이 나옴"

### 3.3 견적 산출 2단계

```
1단계 (최우선): 큰 소재 사이즈 확인 → 여유치 추가 → 소재 비용 산출
2단계 (나중):   구멍 수, 가공 형태 파악 → 가공비 추가 산출
```

김 대표 원문: "1차적으로 큰 소재 사이즈를 확인해서 소재 비용을 먼저 빼내고, 그다음에 잔잔한 구멍들이라든지 이런 것들이 작업이 어떻게 되는가에 따라서 가공비가 추가로 들어가기 시작함"

### 3.4 도면에서 뽑을 것 / 안 뽑을 것

| 영역 | 필요 여부 | 비고 |
|------|----------|------|
| 큰 소재 사이즈 (Ø550 등) | **1순위 필수** | 소재비 산출 핵심 |
| 구멍 수, 가공 형태 | 2순위 | 가공비 산출 (나중) |
| 각도 정보 | 2순위 | 가공 견적 시 참고 |
| 상단 공차 테이블 | 불필요 | "일반적 공차 표시, 견적상 의미 없음" |
| NOTES | 불필요 | |
| ISO VIEW | 불필요 | |
| SECTION 상세 | 2순위 | "한참 뒤에 접근해야 되는 부분" |

### 3.5 작업 범위 확정

| 자료 | 사용 여부 | 이유 |
|------|----------|------|
| BOM | **사용** | 재질, 도면번호, 부품 목록 |
| 단품 도면 (하얀색) | **사용** | 치수/재질 추출 대상 |
| 조립도 (분홍/파란) | 불필요 | 조립 관계 확인용, 치수 없음 |
| 사양서 | 불필요 | BOM에 재질 다 있음 |
| 품질기준서 | 불필요 | 견적과 무관 |
