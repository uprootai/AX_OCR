## POC 테스트 결과 (2026-03-24)

### 1번 자료 (FB2FB6 개별 밸브) — 자동 비교 성공

**입력**: 도면 PDF (pdfplumber) + ERP BOM 엑셀 (zipfile/xml)
**방법**: 텍스트 레이어에서 테이블 추출 → 품번 문자열 매칭 (exact + prefix)
**LLM/YOLO/OCR 사용 없음**

| 비교 결과 | 건수 | 설명 |
|-----------|------|------|
| 완전 일치 | 18건 | 품번+수량 동일 |
| 코드 유사 (불일치) | 5건 | 도면 코드 뒤에 재질/업체 접미사 상이 |
| 도면에만 존재 | 4건 | ERP에서 다른 코드로 등록됨 |
| ERP에만 존재 | 4건 | 도면에서 다른 코드로 등록됨 |

**고객이 수동으로 찾은 7건 중:**
- 코드 유사 5건: STB-M12-50L→193B8MCL2, HN-M12-HV→8M, NAME-PL→KGS, SM→KGS, PS→LIP
- 완전 불일치 4건(쌍): BS-85-6F-T1↔T4, OR-82.22-V↔E, BALL-2,4-304↔2,4M-304, ANSI B18↔HB-M8

**핵심 발견:**
1. 이 도면은 Inventor 출력 벡터 PDF → 텍스트 레이어 있음 → pdfplumber로 100% 추출
2. prefix 매칭만으로 5건 검출, fuzzy matching 추가 시 나머지 4건도 검출 가능
3. **전체 파이프라인: pdfplumber + openpyxl + 문자열 비교 = 끝**

### 도면 이미지 해석 한계 (YOLO 검출 불가)

- 밸브 도면은 **어셈블리(조립) 상태 단면도** → 개별 부품이 결합되어 보이지 않음
- 파나시아(P&ID)와 달리 개별 심볼이 분리되어 있지 않음
- 예: 6페이지(VD-01) 29개 부품 중 일부 번호(29번 등)가 도면 이미지에서 시각적으로 확인 불가
- **결론: YOLO 기반 심볼 검출은 이 도면 유형에 적용 불가**

### 검증 접근법 확정 (통화 합의)
