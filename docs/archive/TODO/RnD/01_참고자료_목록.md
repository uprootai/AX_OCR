### 프로젝트 개요

## 3. 현재 AX 프로젝트 (Microservices 기반)
**경로**: `/home/uproot/ax/poc`

### 프로젝트 개요
- 도면 기반 자동 견적 생성 시스템
- Microservices 아키텍처
- 4개 독립 API 서비스 + Web UI

### 핵심 기술 스택

#### 3.1 eDOCr2 API (OCR 서비스)
- **이중 배포**: v1 (port 5001) + v2 (port 5002)
- **v1**: Keras 2.x, 안정적, 빠른 처리
- **v2**: Keras 3.x, 고급 기능 (테이블 OCR 지원)
- **추출 항목**: 치수, GD&T, 텍스트/정보블록
- **기술**: CRNN (Convolutional Recurrent Neural Network)

#### 3.2 EDGNet API (세그멘테이션 서비스)
- **Graph Neural Network** 기반 도면 세그멘테이션
- **GraphSAGE** 치수 분류기
- **분류 항목**: 윤곽선, 텍스트, 치수
- **정확도**: 90.82%
- **벡터화 지원**: 베지어 곡선 변환

#### 3.3 Skin Model API (공차 분석 서비스)
- 기하 공차 예측
- 제조 가능성 분석
- GD&T 검증

#### 3.4 Gateway API (오케스트레이션)
- 전체 파이프라인 관리
- eDOCr2 → EDGNet → Skin Model 워크플로우
- 견적 생성

#### 3.5 Web UI
- **프레임워크**: React 19 + TypeScript + Vite
- **상태 관리**: Zustand
- **데이터 페칭**: TanStack Query (React Query)
- **스타일링**: Tailwind CSS
- **라우팅**: React Router v7

### 아키텍처 특징
- **Microservices**: 독립적 배포 및 확장
- **Docker 기반**: 컨테이너화된 서비스
- **CORS 설정**: 로컬 개발 지원
- **볼륨 마운트**: 모델/데이터 공유
- **Health Check**: 서비스 상태 모니터링

### 현재 성능
- **전체 파이프라인**: ~25-30초 (CPU)
- **서비스별**: ~8-10초
- **치수 재현율**: ~50% (개선 예정 → 90%)
- **GD&T 재현율**: ~20% (개선 예정 → 75%)

### 활용 가능한 기술 (과제 연관성)
- ✅ Multi-OCR (eDOCr v1/v2 이중 배포)
- 🔄 **VLM 통합** (GPT-4V, Claude Vision 연동 예정)
  - 도면 맥락 이해 및 관계 분석
  - 멀티모달 RAG 시스템
  - 누락 정보 능동적 감지
  - 자연어 견적 설명 자동 생성
  - 기대 효과: 치수 재현율 50% → 90%, GD&T 재현율 20% → 75%
- ✅ 그래프 신경망 기반 세그멘테이션 (GraphSAGE)
- ✅ 마이크로서비스 아키텍처
- ✅ Docker 기반 배포
