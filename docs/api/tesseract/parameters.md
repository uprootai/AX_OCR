# Tesseract OCR API Parameters

> Google Tesseract 기반 범용 OCR 엔진

## 기본 정보

| 항목 | 값 |
|------|-----|
| **Port** | 5008 |
| **Endpoint** | `/api/v1/ocr` |
| **Method** | POST |
| **Content-Type** | multipart/form-data |
| **Timeout** | 60초 |

## 파라미터

### lang
| 항목 | 값 |
|------|-----|
| **타입** | select |
| **기본값** | `eng` |
| **옵션** | `eng`, `kor`, `jpn`, `chi_sim`, `chi_tra`, `eng+kor` |
| **설명** | 인식 언어 |

- **eng**: 영어
- **kor**: 한국어
- **jpn**: 일본어
- **chi_sim**: 중국어 (간체)
- **chi_tra**: 중국어 (번체)
- **eng+kor**: 영어 + 한국어

### psm (Page Segmentation Mode)
| 항목 | 값 |
|------|-----|
| **타입** | select |
| **기본값** | `3` |
| **옵션** | `0`, `1`, `3`, `4`, `6`, `7`, `11`, `12`, `13` |
| **설명** | 페이지 분할 모드 |

| PSM | 설명 |
|-----|------|
| 0 | OSD만 (스크립트 감지) |
| 1 | 자동 OSD |
| 3 | 자동 페이지 분할 (OSD 없음) **기본값** |
| 4 | 가변 크기 텍스트 단일 열 |
| 6 | 단일 균일 블록 |
| 7 | 단일 텍스트 라인 |
| 11 | 희소 텍스트 |
| 12 | OSD 포함 희소 텍스트 |
| 13 | Raw 라인 |

### output_type
| 항목 | 값 |
|------|-----|
| **타입** | select |
| **기본값** | `data` |
| **옵션** | `string`, `data`, `dict` |
| **설명** | 출력 형식 |

- **string**: 단순 텍스트 문자열
- **data**: 구조화된 데이터 (위치 정보 포함)
- **dict**: 딕셔너리 형식

## 입력

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `image` | Image | ✅ | 분석할 이미지 |

## 출력

| 필드 | 타입 | 설명 |
|------|------|------|
| `texts` | OCRResult[] | 인식된 텍스트 목록 |
| `full_text` | string | 전체 텍스트 |

## 사용 팁

- PSM 3은 자동 페이지 분할로 대부분의 문서에 적합
- PSM 6은 단일 블록으로 처리하여 표나 양식에 유용
- OCR Ensemble과 함께 사용시 15% 가중치 권장
