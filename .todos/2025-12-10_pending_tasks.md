## 1. 비용 산정 엔진 (중요도: 중간)

**필요 이유**: 도면 분석 결과를 실제 견적으로 연결해야 고객 가치가 완성됨.

### 1.1 재질 데이터베이스
```
SS400, S45C, SUS304 등 재질별 kg당 단가와 가공성 지수를 저장하는 테이블.
도면에서 추출한 재질명과 매칭하여 원자재 비용을 자동 계산.

예상 작업량: 1-2일
```

### 1.2 공차 등급 테이블
```
ISO IT6~IT18 등급별 가공시간 배수와 추가 비용률 정의.
eDOCr2가 추출한 H7, ±0.01 같은 공차를 파싱하여 등급 판정.
IT6(정밀)은 IT10(일반) 대비 3배 가공시간 필요.

예상 작업량: 1일
```

### 1.3 공정 규칙 엔진
```
치수/공차/표면조도 조합에 따라 필요 공정 자동 판단.
예: "φ10mm H7 구멍 → 드릴+보링+리밍", "Ra 0.8 이하 → 연삭 추가"
필요 공정 목록과 예상 가공시간 반환.

예상 작업량: 2-3일
```

### 1.4 유사 부품 기반 견적
```
Knowledge Engine에서 크기/재질/형상이 유사한 과거 부품 검색.
해당 부품의 실제 견적 데이터를 가중 평균하여 예상 견적 산출.

예상 작업량: 2일
```

## 2. 견적서 출력 (중요도: 중간)

**필요 이유**: CSV만으로는 고객에게 전달하기 부적합. 전문적인 견적서 필요.

### 2.1 Excel 견적서 템플릿
```
회사 로고, 품명/재질/수량/단가 테이블, 소계/VAT/합계 포함.
openpyxl로 템플릿 생성, 분석 결과 자동 채움.

예상 작업량: 2일
```
