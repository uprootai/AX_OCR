## BOM 색깔 체계

```
분홍색 = 최종 조립도 (예: T1 BEARING ASSY)
├── 파란색 = 중간 조립품 (예: BEARING RING ASSY)
│   └── 하얀색 = 단품 ← 견적 대상
└── 하얀색 = 단품 ← 견적 대상
```
