# S02: 뷰별 레이아웃 분리 실험

> GAR 배치도의 여러 뷰(Top/Front/Right/A View)를 분리하여 개별 OCR 수행

| 항목 | 값 |
|------|---|
| 상태 | ✅ Done |
| 의존성 | S01 완료 후 (최적 엔진 선정) |

---

## 실험 내용

### 방법 A: 수동 크롭 → OCR
- 이미지에서 각 뷰 영역을 수동으로 좌표 지정하여 크롭
- 크롭된 개별 뷰에 OCR 적용
- 전체 이미지 OCR 결과와 비교

### 방법 B: 자동 레이아웃 분석
- Surya OCR의 레이아웃 분석 기능 활용
- 또는 OpenCV 기반 영역 분할 (빈 공간/경계선 감지)
- 자동 분리된 뷰에 OCR 적용

### 방법 C: VLM 기반 뷰 인식
- Claude/GPT Vision에 "이 도면에서 각 뷰 영역을 박스로 표시해줘" 요청
- 좌표 기반 크롭 → OCR
