# S03: 대표자 검토 + 기타의견 + 제출

> **Epic**: E01 — 성과발표회 PPT 완성
> **상태**: ⬜ Todo
> **예상**: 30m

---

## 설명

대표자(이정우)가 최종 PPT를 검토하고, 슬라이드 14 "기타 의견"을 직접 작성한다.
검토 완료 후 제출 파일을 확정한다.

## 완료 조건

- [ ] 대표자가 14슬라이드 전체 내용 승인
- [ ] 슬라이드 14 "기타 의견" 텍스트 확정 (주관기관 피드백)
- [ ] 최종 pptx 파일 생성
- [ ] 주관기관 제출 완료 (~03-06)

## 변경 범위

| 파일 | 작업 |
|------|------|
| `docs/초기창업패키지/presentation/generate_pptx.py` | 수정 (기타의견 텍스트 추가) |
| `docs/초기창업패키지/presentation/업루트_성과발표회_최종.pptx` | 최종 생성 |

## 에이전트 지시

```
이 Story는 대표자 입력이 필요합니다.
1. 대표자에게 슬라이드 14 "기타 의견" 작성 요청
2. 제공된 텍스트를 generate_pptx.py에 반영
3. 최종 pptx 생성 + 프리뷰 확인
4. 완료 시: 이 파일의 상태를 ✅ Done으로 변경
```

## 구현 노트

슬라이드 14 포함 내용:
- 주관기관(동아대학교) 피드백
- 창업 프로그램, 멘토링, 공간/장비 지원 관련 의견
- 사업 운영 개선 건의사항
