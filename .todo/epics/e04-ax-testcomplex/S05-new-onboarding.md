# S05: 신규 기업 순차 온보딩

> **목표**: 확정되는 7개사를 S01~S03 체계에 따라 순차 온보딩
> **우선순위**: 낮음 (Blocked — 기업 모집 확정 대기)
> **의존**: S01, S02, S03, S04

---

## 전제 조건

- S01 범용 납품 패키지 완성
- S02 커스터마이징 파이프라인 정립
- S03 교육 체계 구축
- S04 기존 3개사 통합 검증 완료 (체계가 실전에서 동작함을 확인)

## 온보딩 절차 (기업당 6~8주)

S01~S03에서 만든 체계를 그대로 적용:

```
Week 1-2: Phase 1 — 도면 분석
  ├─ 고객 도면 샘플 10장+ 수집
  ├─ 도면 유형 판별 (자동 분류 도구)
  └─ 기존 모델 커버리지 측정 → 리포트

Week 3-4: Phase 2 — 모델 적응
  ├─ YOLO Fine-tuning 또는 전용 파서 생성
  ├─ GT 라벨 생성 → F1 ≥ 90% 검증
  └─ E2E 테스트 작성

Week 5-6: Phase 3 — 납품 및 교육
  ├─ 납품 패키지 생성 (generate-package.sh)
  ├─ 온프레미스 설치 + 동작 확인
  ├─ 교육 실시 (퀵카드 + 매뉴얼 + 영상)
  └─ 운영 지원 SLA 시작

Week 7-8: 안정화
  ├─ 현장 피드백 수집
  ├─ 모델 재학습 (필요 시)
  └─ 최종 인수 확인
```
