# S03: 사용자 교육 체계 구축

> **목표**: 기존 개발자용 문서를 넘어, 현장 엔지니어가 바로 쓸 수 있는 교육 자료 체계 구축
> **우선순위**: 중간 (S01~S02와 병렬 가능)

---

## 현재 상태 (As-Is)

| 자산 | 대상 | 문제점 |
|------|------|--------|
| 설치 가이드 (527줄) | IT 관리자 | ✅ 충분 |
| 운영/관리자 매뉴얼 | IT 관리자 | ✅ 충분 |
| 온보딩 가이드 (MDX) | 개발자 | API 호출 위주, 엔지니어에게 어려움 |
| docs-site 101페이지 | 개발자 | 기술 문서, 사용법 가이드 부족 |
| 시연 영상 (3:55) | 내부용 | 자막 있으나 교육용으로는 너무 빠름 |

**Gap**: 도면을 분석하는 **현장 엔지니어**를 위한 교육 자료가 없음.

## 목표 상태 (To-Be)

### 교육 자료 3종 세트 (고객사별 제공)

| # | 자료 | 형식 | 대상 | 분량 |
|---|------|------|------|------|
| 1 | **빠른 시작 가이드** | PDF (A4 양면 1장) | 엔지니어 | 2페이지 |
| 2 | **사용자 매뉴얼** | 웹 (docs-site) | 엔지니어 | 10~15페이지 |
| 3 | **교육 영상** | MP4 (자막 포함) | 전체 | 10~15분 |

### 빠른 시작 가이드 (퀵카드)
- 로그인 → 프로젝트 선택 → 도면 업로드 → 검출 결과 확인 → 검증(승인/거부/수정) → BOM 다운로드
- 스크린샷 6장 + 화살표 흐름

### 사용자 매뉴얼
- 화면별 기능 설명 (대시보드, 워크플로우, 검증, BOM)
- 자주 묻는 질문 (FAQ) 10개
- 트러블슈팅 (업로드 실패, 검출 누락 시 대처)

### 교육 영상
- 기존 시연 영상(3:55)을 기반으로 편집
- 파트별 분리: ① 시스템 소개(2분) ② P&ID 검출(5분) ③ 기계 도면 치수(5분) ④ BOM 내보내기(3분)

## 작업 항목

| # | 작업 | 산출물 |
|---|------|--------|
| 1 | 빠른 시작 가이드 디자인 + 작성 | PDF 템플릿 |
| 2 | 사용자 매뉴얼 docs-site 섹션 추가 | `docs-site/docs/user-guide/` |
| 3 | FAQ 10개 작성 | FAQ 페이지 |
| 4 | 교육 영상 편집 (파트 분리 + 속도 조절) | MP4 4편 |
| 5 | 고객사별 커스터마이징 포인트 정의 | 템플릿 변수 목록 |

## 수용 기준

- [ ] 빠른 시작 가이드만으로 엔지니어가 도면 1장을 업로드→검증→BOM 다운로드 완료 가능
- [ ] 사용자 매뉴얼에서 모든 화면의 주요 기능이 스크린샷과 함께 설명됨
- [ ] 교육 영상이 파트별로 분리되어 필요한 부분만 시청 가능
