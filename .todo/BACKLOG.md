## 📚 참조 문서

### 새 API 추가 체크리스트

**원본**: `01_NEW_API_CHECKLIST.md`

새 API 추가 시 6단계 체크리스트:

1. **마이크로서비스 생성**
   ```bash
   python scripts/create_api.py {api-name} --port {port} --category {category}
   ```

2. **API 스펙 작성**
   - `gateway-api/api_specs/{api-name}.yaml`

3. **Gateway 통합**
   - Executor 등록
   - 라우터 추가

4. **프론트엔드 통합**
   - nodeDefinitions.ts
   - apiRegistry.ts

5. **테스트 작성**
   - 단위 테스트
   - 통합 테스트

6. **문서화**
   - CLAUDE.md 업데이트
   - API 문서 작성
