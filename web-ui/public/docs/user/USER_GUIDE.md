## 🎯 시스템 소개

### 무엇을 하는 시스템인가요?

공학 도면(PDF/JPG)을 업로드하면 **AI가 자동으로**:
- ✅ 치수를 읽어서 추출
- ✅ 도면을 영역별로 분류
- ✅ 필요한 공차를 계산
- ✅ 제조 견적을 생성

### 누가 사용하나요?

- 📐 **기계 설계자**: 도면 검증 및 치수 확인
- 💼 **견적 담당자**: 빠른 견적 생성
- 🏭 **생산 관리자**: 제조 가능성 검토
- 👥 **영업팀**: 고객 응대용 신속 견적

### 어떤 이점이 있나요?

| 기존 방식 | AI 시스템 | 개선 효과 |
|----------|----------|----------|
| 2-3시간 (수작업) | 30분 (자동화) | **⚡ 80% 시간 절감** |
| 오류 가능성 | 일관된 정확도 | **✅ 95% 정확도** |
| 1명 전담 필요 | 누구나 가능 | **👥 인력 효율화** |

## 🚀 시작하기 (3단계)

### Step 1: 웹 브라우저 열기

아무 브라우저나 열고 주소창에 `http://localhost:5173`을 입력합니다.

**화면이 안 뜨나요?** → [문제 해결](#-문제-해결) 섹션을 참고하세요.

### Step 2: "시작하기" 클릭

홈 화면에서 큰 파란색 버튼 "시작하기"를 클릭합니다.

### Step 3: 도면 업로드

1. "파일 선택" 버튼 클릭
2. 분석할 도면 파일 선택 (PDF, JPG, PNG)
3. "분석 시작" 버튼 클릭
4. 잠시 기다리면 결과 표시 ✅

**그게 다예요!** 정말 간단하죠?

## 💡 주요 기능 사용법

### 기능 1: 치수 자동 추출

**언제 사용?**
- 도면에서 치수를 빠르게 확인하고 싶을 때
- 수작업 측정 검증이 필요할 때

**사용법**:
1. 도면 업로드
2. "치수 추출" 옵션 체크 ✅
3. 분석 시작
4. 결과에서 **"Dimensions"** 탭 확인

**결과 예시**:
시스템이 추출한 치수는 다음과 같은 형식으로 표시됩니다:
- 길이: 150mm (±0.1)
- 지름: Ø50mm (±0.05)
- 각도: 45° (±1°)

각 치수 항목에는 공칭 값과 허용 공차가 함께 표시됩니다.

### 기능 2: 도면 세그멘테이션

**언제 사용?**
- 복잡한 도면에서 영역별 분석이 필요할 때
- 특정 부분만 확대해서 보고 싶을 때

**사용법**:
1. 도면 업로드
2. "세그멘테이션" 옵션 선택
3. 분석 시작
4. 색상별로 구분된 영역 확인:
   - 🔵 **파란색**: 윤곽선 (Contour)
   - 🟢 **초록색**: 텍스트 (Text)
   - 🔴 **빨간색**: 치수 (Dimension)

**활용 팁**:
- 마우스 올리면 상세 정보 표시
- 클릭하면 해당 영역 확대

### 기능 3: 공차 자동 계산

**언제 사용?**
- 제조 전 공차 검토가 필요할 때
- 빠른 제조 가능성 판단이 필요할 때

**사용법**:
1. 도면 업로드
2. **재료 선택** (Steel, Aluminum 등)
3. **공정 선택** (Machining, Casting 등)
4. "공차 계산" 실행
5. 결과 확인:
   - Flatness (평면도)
   - Cylindricity (원통도)
   - Position (위치도)

### 기능 4: 실시간 모니터링

**언제 사용?**
- 시스템 상태 확인이 필요할 때
- 처리 속도가 느릴 때 원인 파악

**사용법**:
1. 우측 상단 "모니터" 아이콘 클릭
2. 서비스별 상태 확인:
   - 🟢 정상
   - 🟡 느림
   - 🔴 오류

## 📚 실전 예제

### 예제 1: 일반적인 기계 부품 도면

**시나리오**: 샤프트 도면 분석

**과정**:
1. 샤프트 도면 업로드 (PDF)
2. 옵션 설정:
   - ✅ 치수 추출
   - ✅ GD&T 인식
   - ✅ 텍스트 추출
3. 분석 시작 → **약 30초 대기**
4. 결과 확인:
   - 치수: 11개 추출됨
   - 지름: Ø50mm, Ø30mm
   - 길이: 200mm

**예상 시간**: 1분

### 예제 2: 복잡한 조립도

**시나리오**: 여러 부품이 포함된 조립도

**과정**:
1. 조립도 업로드
2. **EDGNet 전략** 선택 (고급 옵션)
3. 세그멘테이션 실행 → **약 45초 대기**
4. 결과:
   - 800개 컴포넌트 자동 감지
   - 영역별 색상 구분
   - 각 영역 클릭 가능

**예상 시간**: 2분

### 예제 3: 빠른 견적 생성

**시나리오**: 고객 문의 시 즉시 견적 필요

**과정**:
1. 도면 업로드
2. Gateway API 사용 (전체 파이프라인)
3. 재료/공정 정보 입력
4. "견적 생성" 클릭
5. 자동 생성된 견적서 다운로드

**예상 시간**: 5분

## 🎓 전략 선택 가이드

시스템에는 **4가지 분석 전략**이 있습니다. 상황에 맞게 선택하세요:

### Basic (기본) - 추천 ⭐

**언제?**
- 일반적인 도면 (80% 경우)
- 빠른 처리 필요
- 첫 사용자

**장점**:
- ⚡ 가장 빠름 (~23초)
- 💰 비용 없음
- ✅ 안정적

**단점**:
- GD&T 인식 제한적

### EDGNet - 추천 ⭐⭐

**언제?**
- 복잡한 도면
- 정밀한 분석 필요
- 세그멘테이션 중요

**장점**:
- 📊 800개 컴포넌트 감지
- 🎯 영역별 분류
- 💰 비용 없음

**단점**:
- ⏱️ 처리 시간 약 45초

### VL (Vision-Language) - 고급 ⭐⭐⭐

**언제?**
- 최고 정확도 필요
- GD&T 기호 많은 도면
- 비용 허용

**장점**:
- 🎯 최고 정확도
- ✅ GD&T 70-75% 인식

**단점**:
- 💰 API 비용 발생
- ⏱️ 처리 시간 길음

### Hybrid - 최고급 ⭐⭐⭐⭐⭐

**언제?**
- 중요한 프로젝트
- 완벽한 분석 필요
- 예산 충분

**장점**:
- 🏆 최고 성능
- ✅ GD&T 80-85% 인식
- 📊 완전한 세그멘테이션

**단점**:
- 💰 가장 비쌈
- ⏱️ 가장 느림

**💡 추천**: 처음에는 **Basic**으로 시작, 필요시 **EDGNet** 사용

## ❓ 자주 묻는 질문

### Q1: 도면이 업로드 안 돼요

**A**: 다음을 확인하세요:
- 파일 크기가 50MB 이하인가요?
- 형식이 PDF, JPG, PNG인가요?
- 파일이 손상되지 않았나요?

### Q2: 분석이 너무 느려요

**A**:
- **Basic 전략** 사용 (가장 빠름)
- 도면 크기를 줄이세요 (권장: 2000x2000 픽셀)
- 다른 사용자가 동시에 사용 중일 수 있습니다

### Q3: 치수가 정확하지 않아요

**A**:
- 도면 해상도를 높이세요 (150-300 DPI)
- **EDGNet 전략** 사용
- 도면이 깨끗하고 선명한지 확인

### Q4: GD&T가 인식 안 돼요

**A**:
- 현재 GD&T 인식률: 0-20% (개선 중)
- **VL 전략** 사용 (70-75% 인식)
- 또는 수동으로 입력 필요

### Q5: 결과를 저장하고 싶어요

**A**:
- 결과 화면에서 "다운로드" 버튼 클릭
- JSON, PDF, Excel 형식 선택
- 저장 위치 지정

## 🛠️ 문제 해결

### 문제 1: 웹페이지가 안 열려요

**증상**: `localhost:5173` 접속 불가

**해결**:
터미널에서 `docker ps | grep web-ui` 명령어를 실행합니다.
- 리스트에 web-ui가 없으면 → 시스템을 시작해야 합니다
- 있으면 → 브라우저 캐시를 삭제하고 재시도하세요

### 문제 2: "서비스 오류" 메시지

**증상**: 분석 중 오류 발생

**해결**:
1. 페이지 새로고침 (F5)
2. 다른 파일로 테스트
3. 관리자에게 문의

### 문제 3: 결과가 이상해요

**증상**: 치수가 너무 많거나 이상한 값

**해결**:
1. 도면 품질 확인 (깨끗한가?)
2. 다른 전략 시도 (Basic → EDGNet)
3. 도면 회전/크기 조정 후 재시도

## 🎯 성공 팁

### 빠르게 사용하려면

1. **북마크 등록**: `localhost:5173` 북마크
2. **단축키 외우기**:
   - 업로드: Ctrl+O
   - 분석: Enter
   - 결과 다운로드: Ctrl+S
3. **즐겨찾기 전략**: 보통은 Basic, 중요한건 EDGNet

### 정확도를 높이려면

1. **고품질 도면 사용**:
   - 해상도: 150-300 DPI
   - 크기: 1500x1500 이상
   - 형식: PDF (벡터) > JPG > PNG

2. **전처리**:
   - 회전 보정
   - 대비 향상
   - 노이즈 제거

3. **적절한 전략 선택**:
   - 간단한 도면: Basic
   - 복잡한 도면: EDGNet
   - 중요 프로젝트: Hybrid

### 시간을 절약하려면

1. **배치 처리**: 여러 도면을 한 번에
2. **템플릿 저장**: 자주 쓰는 설정 저장
3. **키보드 사용**: 마우스보다 빠름
