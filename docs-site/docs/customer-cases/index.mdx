# 고객 적용 사례

> AX POC 시스템을 실제 제조 현장에 적용한 고객사별 파이프라인 및 성과를 정리한다.

## 적용 현황

| 고객사 | 업종 | 도면 유형 | 적용 기술 | 상태 |
|--------|------|----------|----------|------|
| **동서기연** | 터빈 베어링 제조 | 기계 부품도 | eDOCr2 치수 추출 + BOM 계층 분류 + 견적 자동 산출 | 개발 중 |
| **파나시아** | 선박 환경장비 | P&ID (MCP Panel) | YOLO 27클래스 심볼 검출 + GT 비교 검증 | POC 완료 |
