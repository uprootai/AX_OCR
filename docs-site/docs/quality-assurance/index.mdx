## 핵심 메트릭 (Key Metrics)

| 메트릭 | 설명 | 목표 |
|--------|------|------|
| **mAP@50** | 평균 정밀도 (Mean Average Precision) | > 0.85 |
| **Precision** | 정밀도 | > 0.90 |
| **Recall** | 재현율 | > 0.85 |
| **F1 Score** | 조화 평균 | > 0.87 |
| **CER** | 문자 오류율 (Character Error Rate, OCR) | < 5% |
