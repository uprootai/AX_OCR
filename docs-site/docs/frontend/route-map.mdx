## 시스템 라우트

| 경로 | 동작 | 설명 |
|------|------|------|
| `/` | `Navigate to="/dashboard" replace` | 루트 접근 시 대시보드로 리다이렉트 |
| `<Layout />` | Outlet wrapper | Header + Sidebar + 페이지 콘텐츠를 감싸는 레이아웃 |

## 라우트 구조

모든 페이지(루트 리다이렉트 제외)는 `<Layout />` 내부에서 렌더링된다. Layout은 Header(내비게이션)와 Sidebar(컨텍스트 의존)를 제공한다.

```
<BrowserRouter>
  <Suspense fallback={<PageLoader />}>
    <Routes>
      /               → Navigate to="/dashboard"
      <Layout>
        /dashboard         → Dashboard
        /guide             → Guide
        /docs              → Docs
        /blueprintflow/builder    → BlueprintFlowBuilder
        /blueprintflow/list       → BlueprintFlowList
        /blueprintflow/templates  → BlueprintFlowTemplates
        /admin             → Admin
        /admin/api/:apiId  → APIDetail
        /pid-overlay       → PIDOverlayPage
        /projects          → ProjectList
        /projects/:projectId → ProjectDetail
      </Layout>
    </Routes>
  </Suspense>
</BrowserRouter>
```
