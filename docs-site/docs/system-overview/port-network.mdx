`http://localhost:3001`은 Docusaurus 문서 사이트이며 기본 `docker-compose.yml`에 포함되지 않습니다.

```bash
cd docs-site
npm install
npm run start   # http://localhost:3001
```

## 헬스체크 엔드포인트(Health Check Endpoints)

모든 서비스는 표준 헬스체크를 제공합니다:

```bash
# Gateway
GET http://localhost:8000/api/v1/health

# 개별 서비스 (서비스별 경로 상이)
# 예: /health, /api/v1/health, /api/v2/health
GET http://localhost:<port>/<health-path>

# BOM 백엔드
GET http://localhost:5020/health
```

| 서비스 | 헬스 엔드포인트 | 예상 응답 |
|--------|----------------|-----------|
| Gateway | `/api/v1/health` | `{"status": "healthy"}` |
| ML API (대부분) | `/health` | `{"status": "healthy"}` |
| 일부 API | `/api/v1/health`, `/api/v2/health` | `{"status": "healthy"}` |
| Neo4j | `:7474` (HTTP) | 브라우저 UI |
