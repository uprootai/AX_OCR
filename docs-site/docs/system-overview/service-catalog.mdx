## 카테고리별 서비스

### 객체 검출(Detection) (2개 서비스)

| 서비스 | 포트 | GPU | 설명 |
|--------|------|-----|------|

| **Tesseract** | 5008 | 미사용 | 문서 OCR, 범용 |
| **TrOCR** | 5009 | 사용 | 필기체 OCR, Transformer 기반 |
| **OCR Ensemble** | 5011 | 사용 | 4엔진 가중 투표, 최고 정확도 |

| **EasyOCR** | 5015 | 미사용 | 80개 이상 언어 지원, CPU 친화적 |
