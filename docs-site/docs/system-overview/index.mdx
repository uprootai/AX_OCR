# System Overview / 시스템 개요

AX POC는 **기계 도면 자동 분석 및 제조 견적 생성 시스템**입니다.

```
도면 이미지 → VLM 분류 → YOLO 검출 → OCR 추출 → 공차 분석 → 리비전 비교 → 견적 PDF
```

## Key Metrics

| Metric | Value |
|--------|-------|
| **Microservices** | 21 Docker containers |
| **API Endpoints** | 24 services |
| **GPU Services** | 10 |
| **CPU Services** | 11 |
| **Detection Classes** | 73 |
| **Node Types** | 29+ |
| **Tests** | 549 (364 backend + 185 frontend) |

## Data Flow

```mermaid
flowchart LR
    IMG["도면 이미지"] --> VLM["VLM 분류"]
    VLM --> |기계도면| YOLO["YOLO 검출"]
    VLM --> |P&ID| PID["P&ID 파이프라인"]
    VLM --> |테이블| TABLE["Table Detector"]

    YOLO --> OCR["OCR 추출"]
    OCR --> DIM["치수 분석"]
    DIM --> TOL["공차 분석"]
    TOL --> BOMB["BOM 생성"]
    BOMB --> QUOTE["견적 PDF"]

    PID --> CONN["연결성 분석"]
    CONN --> DCH["Design Check"]
    DCH --> REPORT["보고서"]
```
