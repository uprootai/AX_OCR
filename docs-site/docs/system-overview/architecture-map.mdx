## Service Communication

모든 서비스 간 통신은 Docker 네트워크 `ax_poc_network`를 통해 이루어집니다.

| Pattern | Description |
|---------|-------------|
| **Gateway → Service** | HTTP REST (JSON/multipart) |
| **Frontend → Gateway** | HTTP REST via Vite proxy |
| **BOM Frontend → BOM Backend** | Direct HTTP |
| **Knowledge → Neo4j** | Bolt protocol (:7687) |
