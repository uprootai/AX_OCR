# 시스템 개요

> AX POC 전체 시스템의 아키텍처, 핵심 지표, 데이터 흐름을 요약한다.

```
도면 이미지 → VLM 분류 → YOLO 검출 → OCR 추출 → 공차 분석 → 리비전 비교 → 견적 PDF
```

## 핵심 지표

| 지표 | 값 |
|------|-----|
| **마이크로서비스(Microservices)** | Docker Compose 서비스 24개 |
| **외부 API 서비스** | 21개 (`docs-site/src/data/services.json` 기준) |
| **Gateway 내부 분석 도구** | 5개 (`titleblock`, `partslist`, `dimensionparser`, `bommatcher`, `quotegenerator`) |
| **GPU 표시 서비스** | 13개 (`apiRegistry.ts`/문서 카탈로그 기준) |
| **검출 클래스(Detection Classes)** | Engineering 14 / P&ID Symbol 32 / BOM 27 |
| **노드 타입(Node Types)** | 29개 이상 |
| **테스트** | 백엔드/프론트엔드 자동화 테스트 스위트 운영 |
