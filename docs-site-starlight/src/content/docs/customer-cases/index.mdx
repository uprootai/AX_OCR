> AX POC 시스템을 실제 제조 현장에 적용한 고객사별 파이프라인 및 성과를 정리했습니다.

| 항목 | 값 |
|------|-----|
| 적용 고객사 | 3개사 |
| 도면 유형 | 기계 부품도, P&ID, 밸브 조립도 |
| 핵심 기술 | eDOCr2, YOLO v11, BOM 자동화, 도면-BOM 교차검증 |

---

## 사례 1 — [동서기연: 터빈 베어링 견적 자동화](./dsebearing)

> 94개 PDF 자료 패키지 → 치수 추출 → BOM 분류 → 견적 자동 산출

| 항목 | 값 |
|------|-----|
| 고객사 | (주)동서기연 — 터빈/발전기 베어링 제조 |
| 전달 자료 | **94개** PDF (BOM 1 + 사양서 2 + 품질기준서 4 + 부품 도면 87) |
| 치수 추출 | **548개** (eDOCr2 + PaddleOCR) |
| BOM 분류 | **326개** (ASSY 7 / SUB 16 / PART 303) |
| 자동 견적 | **₩3,200,466** (11종 54개 품목 기준, 내부 추정 단가) |
| 기존 소요 | 12일 → **사양 확인 4일을 자동화로 대폭 단축** |
| 상태 | 개발 중 (2차 미팅 대기) |

<Aside type="tip" title="핵심 가치">
사양 확인 4일 → eDOCr2 자동 추출로 대체합니다. 도면별 재질·치수 수동 검토를 제거합니다.
</Aside>

---

## 사례 2 — [파나시아: MCP Panel P&ID 심볼 검출](./panasia)

> MCP Panel 도면에서 27종 전력 설비 심볼 자동 검출 → GT 비교 검증 → BOM 생성

| 항목 | 값 |
|------|-----|
| 고객사 | (주)파나시아 — 선박 환경장비 제조 |
| 도면 유형 | P&ID (MCP Panel — 전력 설비 배선도) |
| 검출 심볼 | **27종** (차단기, 변압기, 계전기, 계측기 등) |
| 검증 방식 | GT 비교 + Human-in-the-Loop |
| 납품 형태 | Docker 이미지 4개 (약 9.6GB) |
| 상태 | **POC 완료** |

<Aside type="tip" title="핵심 가치">
YOLO 커스텀 모델로 P&ID 심볼을 자동 검출하고, 독립 실행 Docker 패키지로 납품합니다.
</Aside>

## 사례 3 — [BMT: 도면 & BOM AI검증](./bmt)

> 도면 PDF 부품리스트 자동 추출 → ERP BOM 엑셀 파싱 → 불일치 자동 검출

| 항목 | 값 |
|------|-----|
| 고객사 | (주)비엠티 — 산업용 밸브 제조 (Ball Valve, GVU 등) |
| 도면 유형 | 밸브 조립도 (Inventor), 스키드 GAR, 3D 렌더링 |
| 검증 방식 | pdfplumber 테이블 추출 + 문자열 매칭 (**AI 모델 불필요**) |
| POC 결과 | 도면 27부품 vs ERP 27품목 비교 → 고객 수동 발견 **7건 전부 자동 검출** |
| 상태 | **Phase 0** (자료 분석 완료, 고객 회신 대기) |

<Aside type="tip" title="핵심 가치">
CAD 출력 벡터 PDF는 텍스트 레이어가 살아있어 LLM/YOLO/OCR 없이 pdfplumber + 문자열 비교만으로 검증이 가능합니다.
</Aside>
