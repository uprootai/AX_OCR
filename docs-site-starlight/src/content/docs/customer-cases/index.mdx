# 고객 적용 사례 (Customer Cases)

> AX POC 시스템을 실제 제조 현장에 적용한 고객사별 파이프라인 및 성과를 정리한다.

| 항목 | 값 |
|------|-----|
| 적용 고객사 | 2개사 |
| 도면 유형 | 기계 부품도, P&ID |
| 핵심 기술 | eDOCr2, YOLO v11, BOM 자동화 |

---

## 사례 1 — [동서기연: 터빈 베어링 견적 자동화](./dsebearing)

> 94개 터빈 베어링 도면 → 치수 추출 → BOM 분류 → 견적 자동 산출

| 항목 | 값 |
|------|-----|
| 고객사 | (주)동서기연 — 터빈/발전기 베어링 제조 |
| 도면 | **94개** PDF (1UNIT PJT 전체) |
| 치수 추출 | **548개** (eDOCr2 + PaddleOCR) |
| BOM 분류 | **326개** (ASSY 7 / SUB 16 / PART 303) |
| 자동 견적 | **₩110,747,678** (약 1.1억) |
| 기존 소요 | 12일 → 목표 **1일** |
| 상태 | 개발 중 (2차 미팅 대기) |

<Aside type="tip" title="핵심 가치">
사양 확인 4일 → eDOCr2 자동 추출로 대체. 도면별 재질·치수 수동 검토를 제거한다.
</Aside>

---

## 사례 2 — [파나시아: MCP Panel P&ID 심볼 검출](./panasia)

> MCP Panel 도면에서 27종 전력 설비 심볼 자동 검출 → GT 비교 검증 → BOM 생성

| 항목 | 값 |
|------|-----|
| 고객사 | (주)파나시아 — 선박 환경장비 제조 |
| 도면 유형 | P&ID (MCP Panel — 전력 설비 배선도) |
| 검출 심볼 | **27종** (차단기, 변압기, 계전기, 계측기 등) |
| 검증 방식 | GT 비교 + Human-in-the-Loop |
| 납품 형태 | Docker 이미지 4개 (약 9.6GB) |
| 상태 | **POC 완료** |

<Aside type="tip" title="핵심 가치">
YOLO 커스텀 모델로 P&ID 심볼을 자동 검출하고, 독립 실행 Docker 패키지로 납품한다.
</Aside>
