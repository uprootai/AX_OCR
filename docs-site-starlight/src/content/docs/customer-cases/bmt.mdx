## 프로젝트 개요 (Project Overview)

BMT는 설계 도면(VAULT BOM)과 ERP에 등록된 BOM 간 불일치를 사람이 수동으로 검토하고 있습니다.
AI 검증 솔루션으로 이 과정을 자동화하여 재질 오류, 코드 상이, 수량 불일치를 자동 검출합니다.

## 핵심 발견: LLM/YOLO/OCR 불필요 (Key Finding)

BMT 도면은 Autodesk Inventor에서 출력한 **벡터 PDF**입니다.
텍스트 레이어가 살아있어 `pdfplumber`로 부품리스트 테이블을 100% 추출할 수 있습니다.

| 항목 | 값 |
|------|---|
| 도면 형식 | Inventor .idw → PDF (벡터, 텍스트 레이어 존재) |
| 텍스트 추출 | `pdfplumber` (Python) — 테이블 자동 감지 |
| 엑셀 파싱 | `zipfile` + `xml.etree` (openpyxl 없이 가능) |
| 비교 로직 | exact match + prefix match (문자열 연산) |
| AI 모델 | **사용하지 않음** |

<Aside type="caution" title="전제 조건">
도면이 항상 CAD 출력 벡터 PDF인 경우에만 해당합니다. 스캔본(이미지 PDF)이 들어오면 OCR이 필요합니다. 고객 확인 대기 중입니다.
</Aside>

## 데이터 검증 이슈 (Data Verification Issues)

### SN 시리얼 번호 불일치

<Aside type="caution" title="CRITICAL: 도면 VD 시리얼 불일치">
도면 PDF의 밸브 상세도(VD, 4-13페이지) DOC 번호가 **SN2693**으로, 나머지 자료(GAR/ITDRA/Part List/BOM) 전체의 **SN2709**와 불일치합니다. SN2709용 VD 도면 재요청 완료.
</Aside>

| 섹션 | DOC 번호 | 시리얼 | 상태 |
|------|---------|--------|------|
| GAR (Page 1-3) | W5XGVU-SN2709-GAR | SN2709 | 정상 |
| **VD (Page 4-13)** | W5XGVU-**SN2693**-VD | **SN2693** | **불일치** |
| ITDRA (Page 14-22) | W5XGVU-SN2709-ITDRA | SN2709 | 정상 |
| Part List | W5XGVU-SN2709-PLRA | SN2709 | 정상 |
| ERP BOM | W5XHEGVU-SN2709-DNV | SN2709 | 정상 |

### 도면 이미지 검출 한계

- 밸브 도면은 **어셈블리(조립) 상태 단면도** — 개별 부품이 결합되어 보이지 않음
- 파나시아(P&ID)와 달리 YOLO 기반 심볼 검출 **적용 불가**
- 도면 이미지가 아닌 **도면 내 테이블(Description 표)** 기준으로 검증

### 3번 그룹 = 2번과 동일 파일

- MD5 해시 검증 완료: PDF/Excel 3쌍 모두 동일
- 유일한 추가 파일: 3D 렌더링 참조 이미지 1개

## 매칭 방식 2가지 (Matching Methods)

### 케이스 1: 개별 밸브 — 직접 비교

도면 PDF 부품리스트와 ERP BOM을 품번으로 직접 대조합니다.

```
도면 PDF (pdfplumber)     ERP BOM (엑셀)
┌──────────────────┐     ┌──────────────────┐
│ 항목 | 품번 | 수량 │     │ 순번 | 자품목 | 수량 │
│  3   | BL-85| 1   │ ←→ │  3   | BL-85 | 1   │ ✅
│  4   | BS-T1| 2   │  ✗  │  4   | BS-T4 | 2   │ ⚠️
└──────────────────┘     └──────────────────┘
```

### 케이스 2: 스키드(GVU) — 3단계 매칭

도면 TAG → 파트리스트 CODE → ERP BOM 자품목코드 순으로 매칭합니다.

```
도면 TAG "V01"
  → 파트리스트 CODE "V01" / Drawing No. "FB2F29-R64A-..."
    → ERP BOM 자품목 "FB2F29-R64A-..." / 수량 1EA
```

## 기술 스택 (Tech Stack)

| 레이어 | 기술 | 비고 |
|--------|------|------|
| PDF 추출 | `pdfplumber` (PyMuPDF 백업) | 벡터 PDF 테이블 파싱 |
| 엑셀 파싱 | `zipfile` + `xml.etree` | openpyxl 없이 동작 |
| 비교 엔진 | Python 문자열 매칭 | exact + prefix + fuzzy |
| 리포트 | 엑셀 또는 JSON | 기존 `quotation_excel_exporter` 재활용 가능 |
| OCR (fallback) | `eDOCr2` / `PaddleOCR` | 스캔본 도면 대비용 |
