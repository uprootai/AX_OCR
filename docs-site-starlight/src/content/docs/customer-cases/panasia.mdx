# 파나시아 — MCP Panel (Panasia P&ID Detection)

> MCP Panel 도면에서 27종 전력 설비 심볼을 YOLO로 검출하고, GT 비교 검증 후 BOM을 생성하는 사례.

> **고객**: (주)파나시아
> **업종**: 선박 환경장비 제조 (BWMS, 배기가스 세정장치 등)
> **기간**: 2025 ~ 2026-01 (POC 완료)
> **도면 유형**: P&ID (MCP Panel — 전력 설비 배선도)

---

## 프로젝트 개요

파나시아의 MCP(Motor Control Panel) 도면에서 **27종 전력 설비 심볼**을 자동 검출하고,
Ground Truth(GT) 비교를 통해 검출 정확도를 검증한 뒤, BOM을 생성하는 시스템입니다.

## 검출 대상: 27종 P&ID 심볼

YOLO 모델(`panasia` 프로파일)로 MCP Panel 도면에서 전력 설비 심볼을 검출합니다.

| 카테고리 | 심볼 예시 | 설명 |
|---------|----------|------|
| 차단기류 | CB, MCCB, NFB, ELCB | 회로 차단기 |
| 변압기류 | TR, CT, PT, GPT | 변압기, 계기용 변성기 |
| 보호계전기 | OCR, OCGR, SGR, OVR, UVR | 과전류, 지락, 전압 계전기 |
| 계측기 | AM, VM, WM, WHM, FM | 전류/전압/전력/주파수계 |
| 제어기기 | MC, THR, LA, PF | 전자접촉기, 열동릴레이, 피뢰기 |
| 기타 | DS, VCB, ACB, VS, ZCT | 단로기, 진공차단기 등 |

### 모델 설정

| 항목 | 값 |
|------|---|
| 모델 프로파일 | `panasia` |
| 검출 클래스 | 27종 |
| Confidence Threshold | 0.40 |
| IoU Threshold | 0.50 |
| 이미지 크기 | 1024px |
| SAHI (타일 분할) | 비활성 |

## GT 비교 검증

검출 결과의 정확도를 Ground Truth와 비교하여 검증합니다.

### 검증 워크플로우

1. **YOLO 자동 검출** — 도면에서 심볼 위치·클래스 검출
2. **GT 데이터 비교** — 사전 라벨링된 정답과 매칭
3. **카드 그리드 검토** — 각 검출 결과를 카드로 표시
   - 자동 승인 (높은 신뢰도)
   - 수동 검토 (낮은 신뢰도)
   - 거부/수정 (오검출)
4. **Canvas 수동 추가** — 미검출 심볼 직접 추가
5. **BOM 생성** — 최종 확정된 심볼 목록으로 BOM 내보내기

## 납품 패키지

POC 완료 후 **독립 실행 가능한 Docker 패키지**로 납품합니다.

### 패키지 구성

| 서비스 | 컨테이너 | 포트 | 설명 |
|--------|---------|------|------|
| BOM 프론트엔드 | `panasia-blueprint-ai-bom-frontend` | 13000 | 검증 UI |
| BOM 백엔드 | `panasia-blueprint-ai-bom-backend` | 15020 | 세션/BOM API |
| Gateway API | `panasia-gateway-api` | 18000 | API 게이트웨이 |
| YOLO API | `panasia-yolo-api` | 15005 | 심볼 검출 (GPU) |

### 패키지 메타데이터

```json
{
  "export_version": "3.0",
  "session_name": "PANASIA: MCP Panel 심볼 검출 + GT 비교",
  "container_prefix": "panasia-",
  "port_offset": 10000,
  "network": "panasia_network"
}
```

### 설치 및 실행

```bash
# Linux/macOS
./scripts/import.sh

# Windows PowerShell
.\scripts\import.ps1
```

실행 후 `http://localhost:13000`에서 검증 UI에 접속할 수 있습니다.
