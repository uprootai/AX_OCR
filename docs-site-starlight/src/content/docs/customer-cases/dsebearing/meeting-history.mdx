# 미팅 & 커뮤니케이션 (Meeting & Communication History)

> 동서기연과의 이메일 교환, 1차 미팅 상세 Q&A, 견적 프로세스 확인 내용을 기록한다.

import { Aside } from '@astrojs/starlight/components';

<Aside type="tip" title="1차 미팅 핵심 결정 3가지">
1. **BOM 색깔이 견적 대상을 결정한다** — 하얀색(단품)만 견적 산출, 분홍/파란(조립품)은 스킵
2. **1순위 추출 정보는 큰 소재 치수이다** — 외경·내경·길이를 먼저 뽑아 소재비를 산출
3. **사양서·품질기준서는 견적에 불필요하다** — BOM에 재질 정보가 충분
</Aside>

## 초기 이메일 교환 (01-20 ~ 01-23)

### 업루트 → 동서기연 요청 사항

| # | 요청 | 응답 |
|---|------|------|
| 1 | 도면 샘플 5~10장 | 김 대표가 **94개 전체** 전달 (기대 이상) |
| 2 | 도면에서 추출 필요 정보 | **직접 답변 없음** (미팅 필요) |

| 의제 | 내용 |
|------|------|

| 의제 2 | 도면에서 무엇을 우선 추출해야 하는지 — 재질, 치수, 공차(허용 오차), 표면거칠기, 열처리 |
| 의제 3 | 실제 견적 업무 흐름 파악 — 견적서 양식, 외주 요청서 공유 요청 |

## 1차 미팅 상세 (02-04, 20분, 화상)

**참석**: 업루트 대표 + 윤동민 팀장 + 개발팀원 / 동서기연 김동엽 대표

### 해결된 질문

| 질문 | 김동엽 대표 답변 |
|------|----------------|
| SS400 vs SS275 불일치? | 비슷한 재질이라 대략 적어놓은 것이며 **정확한 스펙은 SS275**이다. 스펙 자료 추가 전달 약속. |
| 사양서/품질기준서가 견적에 필요한가? | 불필요하다. BOM에 재질 정보가 모두 있으므로 그것만으로 충분하다. |
| 조립도(분홍/파란)에서 치수를 추출해야 하는가? | 불필요하다. 조립도는 부품 배치를 보여줄 뿐, 개별 크기는 단품도에 있다. |

### 핵심 발견: BOM 색깔 구분

1차 미팅 최대 성과 — BOM의 색깔이 **견적 대상 여부**를 결정한다는 것을 확인했습니다.

> BOM 색깔별 분류 기준은 [전달 자료 & 도면 분석](/customer-cases/dsebearing/data-overview#bom-색깔-구분)을 참고한다.

:::tip[김동엽 대표 원문]
*"하얀색으로 표기돼 있는 도면들을 개개별로 소재 치수부터 시작해서 단품의 견적이 다 나와야지 합산 견적이 나오거든요."*
:::

:::

### 도면 영역별 추출 필요성 (김동엽 대표 확인)

| 도면 영역 | 필요 여부 | 김 대표 설명 |
|----------|----------|-------------|
| **큰 소재 사이즈** (외경 Ø550 등) | **1순위 필수** | 원재료 주문에 필요한 크기 — 소재비 산출의 핵심 |
| 구멍 수, 가공 형태 | 2순위 (나중) | 가공비 산출에 필요 — 구멍이 많을수록 가공비 증가 |
| 각도 정보 | 2순위 (나중) | 가공 견적 정밀화 시 참고 |
| 가공 표기 (표면처리 등) | 참고 | 특정 부위의 가공 방법 지시 |
| 상단 공차 테이블 | **불필요** | 허용 오차 범위 표시 — 견적에는 영향 없음 |
| NOTES / ISO VIEW | **불필요** | 참고 사항·입체 뷰 — 견적 무관 |
| SECTION 상세 | 2순위 (나중) | 단면 상세도 — 가공 정밀화 단계에서 활용 |

### 작업 범위 확정

| 자료 | 사용 여부 | 이유 |
|------|----------|------|
| BOM | **사용** | 소재 재질, 도면번호, 부품 목록 기준 |
| 단품 도면 (하얀색) | **사용** | 치수, 재질, 가공 정보 추출 대상 |
| 조립도 (분홍/파란색) | **불필요** | *"수치보다는 조립 내용을 표현하는 도면"* |
| 사양서 (STMPS00095) | **불필요** | *"BOM에 재질 다 있으니까 그걸로 충분"* |
| 품질기준서 (TD0060732) | **불필요** | *"견적 볼 때는 안 봐도 되는 부분"* |
