## 카테고리별 분석

### 87개 도면 분류 (추정)

| 카테고리 | 도면 수 | All-3 비율 | 특성 |
|---------|:---:|:---:|------|
| ASSY (조립 베어링) | ~7 | 높음 | OD/ID/W 명확, GT 가능 |
| 단품 (RING, PAD 등) | ~50 | 중간 | OD/ID/W 적용 가능하나 GT 없음 |
| 비베어링 (PIN, WASHER 등) | ~25 | 낮음 | OD/ID/W 개념 부적합 |
| 완전 실패 | ~5 | 0% | OCR 자체 실패 |

### 비베어링 도면 식별

soft filter의 `title_missing_bearing` penalty로 비베어링 도면을 식별한다. 아래는 도면 제목에서 확인된 비베어링 부품 유형이다.

| 유형 | 도면 예시 | 제목 키워드 | 비고 |
|------|---------|-----------|------|
| 케이싱 | TD0060700 | CASING | OD=9458 등 비현실적 값 |
| 핀/키 | TD0060707 | LOCKING PIN | 원형 단면이나 베어링 아님 |
| 와셔 | TD0060720 | WASHER | OD/ID는 존재하나 W 무의미 |
| 시일 | TD0060710 | SEAL RING | 단순 링 부품 |
| 노즐/커버 | TD0060730 | NOZZLE | 비회전 부품 |

**식별 기준** (soft filter):
1. 도면 제목에 `BEARING`, `BRG`, `베어링` 없음 → `title_missing_bearing` (-15%)
2. OD가 50~2000mm 범위 밖 → `od_out_of_range` (-20%)
3. OD ≤ ID → `invalid_physical_order` (-25%)

3개 penalty가 모두 적용되면 confidence가 `1.0 × 0.85 × 0.80 × 0.75 = 0.51`로 떨어진다. **confidence < 0.6인 결과는 비베어링으로 간주**하고 참고용으로만 활용한다.
