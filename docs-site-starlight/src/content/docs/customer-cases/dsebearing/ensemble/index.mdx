## 핵심 메트릭 비교

| 지표 | v5 (GT 2건) | v6 (GT 7건) | **v7 (GT 7건)** |
|------|:-----------:|:-----------:|:---------------:|
| OD | 2/2 | 5/7 | **7/7** |
| ID | 2/2 | 0/7 | **7/7** |
| W | 2/2 | 5/7 | 5/7 |
| **합계** | 6/6 (100%) | 10/21 (48%) | **19/21 (90%)** |

v5의 "100%"는 GT 2건에 한정된 수치였다. GT를 7건으로 확대하자 실제 정확도(48%)가 드러났고, v7에서 SEC 노이즈 필터 + OD 교차검증을 적용하여 **90%**로 개선했다.

:::note[v7 회귀 테스트]
Non-GT 80개 도면 전수 테스트: 에러 0건, OOM 0건, 물리적 순서 위반 2건(소형 부품). 기존 도면에 대한 회귀 없음.
:::

## 왜 All-3은 동일한가?

All-3은 "OD, ID, W 세 값이 모두 존재하는가"만 측정한다.
v2에서 이미 62개 도면이 값을 반환했으므로, 정확도 개선은 이 수치에 영향을 주지 않는다.

나머지 25개 도면은 대부분 **비베어링 부품**(핀, 와셔, 케이싱 등)으로 OD/ID/W 개념 자체가 적용되지 않는다.

## v5에서 적용한 수정 (5개)

| # | 수정 | 대상 파일 | 효과 |
|---|------|-----------|------|
| 1 | `_extract_num()` 파싱 강화 | `dimension_ensemble.py` | 5개 K→k_priority 승격 |
| 2 | Fallback OCR | `dimension_ensemble.py` | K 실패 31개 도면에 S01/S02 공급 |
| 3 | W radial 공식 우선 | `geometry_guided_extractor.py` | TD0062055 W=250→48 수정 |
| 4 | NM/토크 필터 | `geometry_guided_extractor.py` | 342NM 등 비치수 제외 |
| 5 | Non-bearing soft filter | `dimension_ensemble.py` | confidence 기반 이상치 표시 |

각 수정의 상세는 [코드 수정 상세](./code-fixes/) 참조.
