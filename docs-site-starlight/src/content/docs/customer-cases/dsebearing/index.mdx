# 동서기연 — 터빈 베어링 (DSE Bearing Quotation)

> 터빈 베어링 1세트 견적을 12일에서 1일로 줄이기 위한 도면 분석·견적 자동화 사례.

## 고객

| 항목 | 내용 |
|------|------|
| 회사명 | (주)동서기연 (김동엽 대표) |
| 업종 | 터빈/발전기 베어링 제조 |
| 주요 거래처 | 두산에너빌리티 |
| 기간 | 2026-01 ~ 진행 중 |
| 도면 유형 | 기계 부품도 (ISO/JIS 규격) |

<Aside type="note" title="용어 안내">
- **BOM** (Bill of Materials) — 부품 명세서. 제품을 구성하는 모든 부품의 목록
- **ASSY / SUB / PART** — BOM 계층. ASSY(최종 조립품) → SUB(중간 조립품) → PART(개별 단품, 견적 대상)
- **치수** — 도면에 기재된 크기 정보 (외경, 내경, 길이 등)
- **eDOCr2** — AX POC의 OCR 엔진. 도면에서 텍스트와 치수를 자동 추출
</Aside>

---

## 문제: 1세트 견적에 12일

터빈 베어링 1세트를 만들려면 수백 개의 부품이 필요하다. 기존에는 도면을 하나하나 열어 크기를 확인하고, 재질에 맞는 원재료 가격을 조회한 뒤, 가공비를 더해 견적을 산출했다.

| 단계 | 소요일 | 내용 |
|------|--------|------|
| 사양 확인 | 4일 | 도면마다 재질과 크기를 눈으로 확인 |
| 외주 견적 | 6일 | 협력사에 견적 요청 후 취합 |
| 자체 검토 | 2일 | 산출 결과를 사람이 최종 확인 |
| **합계** | **12일** | |

<Aside type="tip" title="김동엽 대표 원문">
*"사양 확인하고 도면에 큰 치수에 여유치를 붙여서 일단 소재 견적을 받아야 되는 부분이거든요."*

**풀이**: 원재료(소재)를 주문할 때는 도면의 최종 크기보다 약간 크게 주문한다. 이 추가분을 **여유치**라고 한다. 가공 중 깎아내는 양을 미리 감안하는 것이다.
</Aside>

| 2026-01-23 | 후속 이메일 (분류확인 + 미팅요청) | ✅ 완료 |
| 2026-02-04 | **1차 미팅** (20분, 화상) — [상세](./meeting-history) | ✅ 완료 |
| 2026-03-04 | 2차 미팅 준비자료 작성 | ✅ 완료 |

| 미정 | **2차 미팅** — 결과 시연 + 단가표 수령 | ⏳ 일정 대기 |

### 핵심 수치

| 지표 | 값 | 비고 |
|------|-----|------|
| 수령 자료 | **94개** PDF | 터빈 베어링 1세트 전달 자료 패키지 |
| BOM 품목 | **326개** (조립품 23 + 단품 303) | 견적 대상은 단품 303개 |
| 치수 추출 | **548개** | 11종 단품 도면 |
| 견적 산출 | **₩3,200,466** | 11종 54개 품목 기준, 내부 추정 단가 |
| 자동화 효과 | **사양 확인 4일 → 자동** | 12일 중 4일 제거 |

<Aside type="tip" title="숫자 읽는 법">
**94개 자료 패키지** 수령 (BOM 1 + 사양서 2 + 품질기준서 4 + 부품 도면 87) → BOM에서 **326개 품목** 식별 → 견적 대상 단품 **303개** → 도면 확보된 **11종**에서 치수 **548개** 추출 → **54개** 품목 견적 산출 완료. 나머지 249개는 도면 미확보 상태.
</Aside>
