# 동서기연 — 터빈 베어링 (DSE Bearing Quotation)

> 94개 터빈 베어링 도면에서 크기 정보를 자동으로 읽어내고, 부품 목록을 계층별로 분류한 뒤, 제조 비용을 자동 산출하는 사례.

터빈 베어링 1세트를 만들려면 수백 개의 부품이 필요하다. 기존에는 도면을 하나하나 열어 크기를 확인하고, 재질에 맞는 원재료 가격을 조회한 뒤, 가공비를 더해 견적을 산출했다. 이 과정에 **12일**이 걸렸다. AX POC는 도면에서 크기 정보를 자동 추출하고, 부품 목록(BOM)을 분류하여 **1일 이내** 견적 산출을 목표로 한다.

<Aside type="note" title="용어 안내">
- **BOM** (Bill of Materials) — 부품 명세서. 제품을 구성하는 모든 부품의 목록
- **ASSY / SUB / PART** — BOM 계층. ASSY(최종 조립품) → SUB(중간 조립품) → PART(개별 단품, 견적 대상)
- **치수** — 도면에 기재된 크기 정보 (외경, 내경, 길이 등)
- **견적** — 제조 예상 비용 산출 (재료비 + 가공비)
- **eDOCr2** — AX POC의 OCR 엔진. 도면에서 텍스트와 치수를 자동 추출
</Aside>

> **고객**: (주)동서기연 (김동엽 대표)
> **업종**: 터빈/발전기 베어링 제조 (주요 거래처: 두산에너빌리티)
> **기간**: 2026-01 ~ 진행 중
> **도면 유형**: 기계 부품도 (ISO/JIS 규격 기반)

### 핵심 성과 지표

| 지표 | 값 | 비고 |
|------|-----|------|
| 수령 도면 | **94개** PDF | 터빈 베어링 1세트 전체 도면 |
| 자동 추출 치수 | **548개** | 11종 단품 도면에서 크기 정보 추출 |
| BOM 품목 분류 | **326개** (조립품 23 + 단품 303) | 견적 대상은 단품(PART) 303개 |
| 자동 산출 견적 | **₩110,747,678 (약 1.1억)** | 54개 품목 기준 (내부 추정 단가) |
| 기존 수작업 소요 | **12일** → 자동화 목표 **1일** | 사양확인 4일 + 외주 6일 + 검토 2일 |

<Aside type="tip" title="숫자 읽는 법">
**94개 도면** 수령 → BOM에서 **326개 품목** 식별 → 이 중 견적이 필요한 개별 단품은 **303개** → 현재 도면이 확보된 **11종**에서 치수 **548개** 추출 → 해당하는 **54개** 품목의 견적 산출 완료. 나머지 249개는 도면 미확보 또는 추가 분석이 필요한 상태이다.
</Aside>

## 고객 Pain Point

동서기연은 터빈 베어링 **1세트 견적에 12일**이 소요된다.

| 단계 | 소요일 | 내용 | AX POC 자동화 |
|------|--------|------|----------------|
| 사양 확인 | 4일 | 도면마다 재질과 크기를 눈으로 확인 | **✅ 완료** — 도면에서 치수 자동 추출 |
| 외주 견적 | 6일 | 협력사에 견적 요청 후 취합 | ⬜ 향후 자동화 예정 |
| 자체 검토 | 2일 | 산출 결과를 사람이 최종 확인 | **✅ 완료** — 검증 UI에서 카드별 확인 |

<Aside type="tip" title="김동엽 대표 원문">
*"사양 확인하고 도면에 큰 치수에 여유치를 붙여서 일단 소재 견적을 받아야 되는 부분이거든요."*

**풀이**: 원재료(소재)를 주문할 때는 도면의 최종 크기보다 약간 크게 주문한다. 이 추가분을 **여유치**라고 한다. 가공 중 깎아내는 양을 미리 감안하는 것이다.
</Aside>

## 시스템 커버리지

현재 AX POC가 동서기연 프로젝트에서 실제로 커버하는 범위이다.

### 자동화 완료 (12일 → 1일 목표 중 "사양 확인 4일" 제거)

| 기능 | 설명 | 인식 패턴 |
|------|------|----------|
| **BOM 색상 기반 분류** | PDF의 행 배경색으로 ASSY/SUB/PART 자동 분류 | 분홍·파란·하양 |
| **타이틀 블록 파싱** | 도면번호(TD0062017), 리비전, 품명, 재질, 스케일 추출 | 정규식 기반 |
| **부품 리스트 파싱** | 도면 내 부품 목록 자동 추출 (4단계 폴백) | 표→키워드→패턴→단순 |
| **치수 자동 추출** | 외경, 내경, 길이, 나사, ISO 공차, GD&T 등 17종 | eDOCr2 + PaddleOCR |
| **견적 자동 산출** | 재료비 + 가공비 + 스크랩비 + 셋업비 + 검사비 + 운송비 | 7개 비용 항목 분리 |
| **OCR 오류 보정** | DSE 재질코드 자동 교정 (SFA4OA → SF440A 등) | 도메인 특화 사전 |
| **검증 UI** | 카드 그리드에서 추출 결과 승인/수정/거부 | 정확도 90%+ 자동 승인 |
| **견적서 내보내기** | PDF + Excel 다운로드 | reportlab + openpyxl |

### 아직 자동화되지 않은 영역

| 영역 | 현재 상태 | 필요 작업 |
|------|---------|----------|
| 외주 견적 (6일) | 수동 프로세스 | 협력사 연동 자동화 |
| 가공비 세부 산출 | 부품 유형별 고정 단가 | 구멍 수·가공 형태별 세부 단가 |
| 고객 실단가 반영 | 내부 추정 21종 | 고객사 단가표 수령 후 교체 |
| 전체 303개 PART 견적 | 54개만 산출 (17.8%) | 나머지 도면 확보 또는 유사 품목 추정 |
