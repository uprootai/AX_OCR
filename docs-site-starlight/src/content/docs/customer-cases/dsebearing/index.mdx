## 고객 Pain Point

동서기연은 터빈 베어링 **1UNIT 견적에 12일**이 소요됩니다.

| 단계 | 소요일 | 내용 | AX POC 커버리지 |
|------|--------|------|----------------|
| 사양 확인 | 4일 | 도면별 재질/치수 수동 검토 | **✅ 자동화 완료** (eDOCr2 + PaddleOCR) |
| 외주 견적 | 6일 | 협력사에 견적 요청·취합 | ⬜ 향후 자동화 |
| 자체 검토 | 2일 | 최종 검토 및 조정 | **✅ 검증 UI** (카드 그리드) |

:::tip[김동엽 대표 원문]
*"사양 확인하고 도면에 큰 치수에 여유치를 붙여서 일단 소재 견적을 받아야 되는 부분이거든요."*
:::
