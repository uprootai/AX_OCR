# 전달 자료 & 도면 분석 (Data Overview & Drawing Analysis)

> 동서기연으로부터 수령한 94개 PDF 자료 패키지의 상세 분류, BOM 구조, 실제 도면 캡쳐를 정리한다.

## 파일 통계 (94개 PDF)

| 분류 | 수량 | 용량 | 내용 | 견적 활용 |
|------|------|------|------|----------|
| **BOM** | 1개 | 0.4MB | 부품 명세서 — 395개 품목의 재질·도면번호 목록 | **사용** — 재질·부품 매칭 기준 |
| **제조사양서** | 1개 | 33MB | 전체 조립도 (부품 간 배치 관계) | 불필요 — 조립 참고용 |

| **품질기준서** | 4개 | - | 품질 검사 기준 문서 (TD0060732) | 불필요 — 견적과 무관 |
| **부품 도면** | 87개 | ~100MB | 개별 부품의 형상·치수 도면 (조립도/단품도 혼재) | **사용** — 치수 추출 대상 |
| **합계** | **94개** | **~145MB** | 터빈 베어링 1세트 전달 자료 패키지 | - |

## 부품 도면 카테고리 (87개)

87개 부품 도면에는 **조립도·중간 조립도·단품도가 혼재**되어 있다. 1차 미팅에서 단품도 정확 개수는 확정되지 않았으며, 견적 대상은 **하얀색 단품도만** 해당한다.

| 카테고리 | 수량 | 설명 | 분석 상태 |
|---------|------|------|----------|
| **체결 부품** (볼트, 너트, 핀, 와셔) | 15개 | 부품을 서로 고정하는 표준 부품 | ✅ 전체 분석 |
| **저널 베어링** (T1~T8) | 40개 | 회전축을 옆에서 지지하는 베어링 — 축 8개 위치 (조립도 포함) | ✅ 전체 분석 |
| **스러스트 베어링** | 12개 | 축 방향(앞뒤) 힘을 받는 베어링 (조립도 포함) | ✅ 전체 분석 |
| **조정 부품** (심, 웨지, 피벗) | 14개 | 정밀 위치 조정용 부품 | ✅ 전체 분석 |
| **기타 부품** | 6개 | 마모 패드, 노즐, 커버 등 | ✅ 전체 분석 |

## BOM 상세 (Z24018)

| 항목 | 값 |
|------|---|
| Project NO | Z24018 |
| Project명 | PP12 |
| DOC NO | BOM_Z24018_110104001 |
| REV | 1 (2025.05.16, PT번호 변경/소재변경) |
| Excel 원본 행 수 | 395개 |
| 시스템 등록 품목 | **326개** (ASSY 7 + SUB 16 + PART 303) |
| 총 중량 | 약 20,698 kg |
| 작성자 | 방경보 |

<Aside type="note" title="395개 vs 326개">
Excel BOM 원본에는 395개 행이 있으나, 빈 행·중복·참조 행 등을 제외하고 실제 계층 분류가 가능한 품목은 **326개**이다. 견적 대상은 이 중 하얀색(PART) **303개**이다.
</Aside>

### BOM 색깔 구분 (1차 미팅에서 확인)

색깔은 **Excel BOM 파일(`Z24018_110104001_BRG_R1`)의 행 배경색**이다. 동서기연 설계팀이 조립 계층에 따라 수동으로 지정한다.

| Excel 행 배경색 | BOM 계층 | 수량 | 견적 처리 |
|---------------|---------|------|----------|
| 🟨 노란색 | ASSY — 최종 조립품 | 7개 | **스킵** — 하위 단품 합산이 이 금액 |
| 🟨 노란색 | SUB — 중간 조립품 | 16개 | **스킵** — 하위 단품 합산 |
| ⬜ **하얀색** | **PART — 개별 단품** | **303개** | **견적 대상** — 각각 소재비+가공비 산출 후 합산 |

### BOM 계층 예시: T1 BEARING ASSY

```
BOM 2번: T1 BEARING ASSY (노란) ← 조립도, 견적 불필요
├── 3번: BEARING RING ASSY (노란) ← 중간 조립품, 견적 불필요
│   ├── 4~17번: 개별 부품 (하양) ← 각각 견적 산출
│   └── 18번: 하위 조립품 (노란) ← 견적 불필요
│       └── 19~23번: 부품 (하양) ← 각각 견적 산출
├── 24~44번: 나머지 부품들 (하양) ← 각각 견적 산출
└── 단품 견적 전부 합산 = 2번 전체 견적
```

## 주요 재질 정리

재질(소재의 종류)에 따라 kg당 단가가 달라지므로, 정확한 재질 파악이 견적의 핵심이다.

| 분류 | 대표 규격 | 용도 | 비고 |
|------|----------|------|------|
| **단조강** | SF440A | 베어링 링, 케이싱, 패드 | 주력 소재 — 대부분의 부품에 사용 |
| **합금강** | SCM440, S45C | 볼트, 핀, 웨지 | 열처리(담금질+뜨임)로 강도를 높인 소재 |
| **스테인리스강** | SUS304, SUS316 | 심 플레이트, 와셔 | 부식 방지가 필요한 부품 |
| **일반 구조용강** | SS275 | 심/로킹 플레이트 | BOM에 SS400으로 기재되었으나 **정확한 스펙은 SS275** (미팅 확인) |
| **특수 재질** | ASTM B23 GR.2 | 베어링 패드 라이닝 | 화이트메탈 — 마찰 감소용 특수 합금 |

## AX POC 시스템 커버리지

### 도면 영역별 자동화 현황

| 도면 영역 | 커버리지 | 사용 기술 | 비고 |
|----------|---------|----------|------|
| **큰 소재 사이즈** (Ø550 등) | ✅ 자동 추출 | eDOCr2 + PaddleOCR | 1순위 — *"소재 치수가 제일 1번으로 빠져야 됨"* |
| **재질 정보** | ✅ BOM 연동 | BOM 파서 | BOM에서 재질 자동 매칭 |
| 구멍 수, 가공 형태 | ⬜ 2단계 예정 | YOLO 검출 | *"가공비가 추가로 들어감"* |
| 각도 정보 | ⬜ 2단계 예정 | eDOCr2 | *"2차로 가공 견적 뺄 때 참고"* |
| 가공 표기 (표면처리) | ⬜ 향후 | OCR | *"가공을 어떻게 해라는 표기들"* |
| 상단 공차 테이블 | ➖ 불필요 | - | *"견적상 크게 의미 없음"* |
| NOTES / ISO VIEW | ➖ 불필요 | - | 확인 결과 불필요 |

### 자료 유형별 사용 여부

| 자료 | 사용 | 이유 |
|------|------|------|
| BOM | ✅ **사용** | 소재 재질, 도면번호, 부품 목록 기준 |
| 단품 도면 (하얀색) | ✅ **사용** | 치수, 재질, 가공 정보 추출 대상 |
| 조립도 (노란색) | ➖ 불필요 | *"수치보다는 조립 내용을 표현하는 도면"* |
| 사양서 (STMPS00095) | ➖ 불필요 | *"BOM에 재질 다 있으니까 그걸로 충분"* |
| 품질기준서 (TD0060732) | ➖ 불필요 | *"견적 볼 때는 안 봐도 되는 부분"* |
