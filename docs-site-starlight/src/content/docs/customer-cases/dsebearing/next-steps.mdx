## 2차 미팅 계획

### 미팅 의제

| # | 의제 | 업루트 준비 상황 |
|---|------|----------------|
| 1 | 소재 사이즈 추출 테스트 결과 공유 | ✅ 완료 (548개 치수) |
| 2 | 다른 형태 도면 (스러스트 베어링 등) 추가 확인 | ✅ 완료 (TD0062057~066, 8종 310개) |
| 3 | BOM 계층 구조 자동화 방안 논의 | ✅ 완료 (326개 분류됨) |

### 시연 화면 구성

1. **BOM 계층 탭** → ASSY/SUB/PART 색깔별 분류 확인
2. **세션 목록** → 도면 클릭 → 치수 카드 그리드 (저널 3종 + 스러스트 8종)
3. **견적 현황 탭** → ₩110,747,678 자동 산출

## 미해결 질문

| # | 질문 | 배경 설명 | 상태 |
|---|------|----------|------|
| 1 | T1, T2 BEARING ASSY가 BOM에 없는 이유 | BOM 목록과 도면 목록이 일부 불일치 — 조립품 번호가 BOM에서 누락된 것인지, 다른 코드로 등록된 것인지 확인 필요 | ⏳ 미확인 |
| 2 | 외주/자체제작 구분 기준 | 부품마다 협력사에 외주하는 것과 자체 제작하는 것의 구분이 있어야 견적 경로가 달라짐 | ⏳ 미확인 |
| 3 | 여유치 산정 기준 (몇 mm? 몇 %?) | 원재료 주문 시 가공 여분을 얼마나 잡는지에 따라 소재비가 달라짐 | ⏳ 미확인 |
| 4 | 가공비 세부 기준 | 구멍 1개당 단가, 가공 난이도별 요율 등 — 2단계 견적 정밀화에 필요 | ⏳ 미확인 |
