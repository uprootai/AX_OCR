# 추출 & 견적 결과 (Extraction & Quotation Results)

> 대표 단품 11종에서 치수 추출과 초안 견적이 가능함을 검증한 단계이다. 전체 자동 견적 완성이 아닌 **1차 검증 결과**이다.

<Aside type="caution" title="현재 결과의 범위">
이 페이지의 금액은 **고객사 단가표 없이 내부 추정 기준**으로 계산한 예비 수치이다. 전체 303개 단품 중 54개(17.8%)만 산출된 상태이며, 고객사 단가표 수령 후 재산출 예정이다.
</Aside>

## 견적 자동 산출

### 시스템 실측 결과 (2026-03-12 Playwright 검증)

프로젝트 대시보드에서 실제 확인된 견적 데이터이다.

| 항목 | 금액 | 비고 |
|------|------|------|
| 총 재료비 | ₩503,326 | 4개 재질 (SCM440, SS400, SUS304, SUS316L) |
| 총 가공비 | ₩1,489,750 | 부품 유형별 26종 단가 적용 |
| 소계 | ₩2,909,514 | |
| **합계 (VAT 포함)** | **₩3,200,466** | 11개 품목, 33개 부품 기준 |

<Aside type="caution" title="이전 산출값과의 차이">
이전에 기재된 ₩110,747,678은 초기 내부 추정값이다. 현재 시스템에 반영된 가격 데이터베이스(20종 재질, 26종 가공비)로 재산출한 결과 ₩3,200,466이다. 고객사 실단가 수령 후 다시 변경될 수 있다.
</Aside>

> **산출 기준**: 내부 추정 단가 기준이며 고객사 단가표 미수령 상태이다. 재료비는 `(소재 외경 × 길이 × 재질별 kg 단가)`로 산출했고, 가공비는 부품 형상 복잡도에 따른 추정치이다. 고객사 단가표 수령 후 재산출 예정.

### 견적 산출 항목 상세

시스템이 각 부품별로 자동 산출하는 비용 항목이다.

| 비용 항목 | 설명 | 산출 방식 |
|----------|------|----------|
| **재료비** | 원재료 구매 비용 | 소재 외경 × 길이 → 체적 → 무게 × 재질별 kg 단가 |
| **가공비** | 절삭·구멍 등 가공 비용 | 부품 유형별 기본 단가 (RING 8만, PAD 5만, BEARING 10만원) |
| **열처리/표면처리** | 담금질, 표면 코팅 비용 | 재질별 열처리 요건에 따라 산출 |
| **스크랩비** | 절삭 폐기물 비용 | 재료비의 약 7.4% (절삭 손실분) |
| **셋업비** | 가공 준비 비용 | 품목당 고정 50,000원 |
| **검사비** | 완성 부품 검사 | 소계의 약 3% |
| **운송비** | 물류 비용 | 소계의 약 0.3% |

<Aside type="note" title="난이도 보정">
현재 모든 DSE 베어링 부품에 **난이도 계수 1.3**이 적용되어 있다. 이는 터빈 베어링이 일반 기계 부품보다 정밀도 요구가 높기 때문이다. 또한 소재 주문 시 가공 여분(여유치) **5mm**가 외경에 자동 추가된다.
</Aside>

## 실제 구현 현황 (Playwright 검증 완료)

2026-03-12 기준, 실제 운영 시스템에서 Playwright 자동 테스트로 검증한 결과이다.

### 구현 완료 — API 테스트 통과

| 기능 | 검증 상태 | 검증 방법 |
|------|----------|----------|
| BOM PDF 파싱 | ✅ 통과 | 프로젝트 상세에서 326개 품목 확인 |
| eDOCr2 치수 추출 | ✅ 통과 | POST /dimensionparser → 3개 치수 + GD&T |
| 타이틀 블록 파싱 | ✅ 통과 | POST /titleblock → TD0062016, SF45A |
| 부품 리스트 파싱 | ✅ 통과 | POST /partslist → 5개 부품 (RING UPPER 등) |
| BOM 매칭 | ✅ 통과 | POST /bommatcher → confidence 0.9 |
| 견적 자동 산출 | ✅ 통과 | POST /quotegenerator → ₩525,844 |
| 견적 대시보드 | ✅ 통과 | 어셈블리별·재질별 분류, 진행률 100% |
| PDF 내보내기 | ✅ 통과 | POST /quote/export/pdf → 200 |
| OCR 오류 보정 | ✅ 통과 | DSE 재질코드 사전 탑재 확인 |
| 고객별 할인 프로필 | ✅ 통과 | GET /customers → 8개 고객사 (기존 6개 → 8개로 확대) |

### 수정 필요 — 테스트 실패

| 기능 | 상태 | 증상 | 필요 조치 |
|------|------|------|----------|
| Excel 내보내기 | ❌ 500 | POST /quote/export/excel → Internal Server Error | 서버 로그 확인 후 수정 |
| BOM 프론트엔드 | ❌ 빈 화면 | 모든 경로에서 SPA fallback 실패 | nginx try_files 설정 |
| 세션 상세 보기 | ❌ 라우트 없음 | `/bom/workflow` 라우트가 web-ui에 미등록 | BOM 앱 연동 수정 |

| 영역 | 현재 상태 | 필요 개선 |
