### 필터링 규칙 (통화 확인)

박순근 팀장님: "V나 P나 T나 F나 이렇게 시작하는 거는 약어인 거예요"

| 패턴 | 분류 | 예시 | 정규식 |
|------|------|------|--------|
| `V + 숫자` | TAG (밸브) | V01, V11-1 | `^V\d+(-\d+)?$` |
| `PT + 숫자` | TAG (압력센서) | PT07 | `^PT\d+$` |
| `TT + 숫자` | TAG (온도센서) | TT01 | `^TT\d+$` |
| `FT + 숫자` | TAG (유량계) | FT01 | `^FT\d+$` |
| `PI + 숫자` | TAG (압력지시계) | PI02 | `^PI\d+$` |
| `B + 숫자` | TAG (필터) | B01 | `^B\d+$` |
| 숫자로 시작 | **치수 (제거)** | 2170, 1450 | `^\d` |
| GVU/WORKSPACE 등 | **노이즈 (제거)** | GVUIN110... | 15개 패턴 |

### 필터링 결과

| 항목 | 건수 |
|------|------|
| OCR raw (S02 5개 뷰 합산) | 39개 |
| **필터 후 TAG** | **24개** |
| 제거된 노이즈 | 15개 |

**제거된 노이즈 15개**:

`A1,B1` `A1,B1,D1,B2,D2` `D1` `D2` `DN1OOGASVALV` `ENCLOSUREDES1GNPRESSURE1MI` `GVUIN110OPENPOSITION` `GVUIN90OPENPOSITION` `GVUSPEC1F1CA` `M8` `MED1AFORENGINE` `MED1AFORPURGE` `NORKSPACE1300X3` `R614` `WORKSPACE800X460`

<Aside type="note" title="Precision 100%">
필터링 후 24개 TAG 전부가 실제 TAG (false positive 0). 노이즈가 완전히 제거되었습니다.
</Aside>

## S04: Part List → ERP BOM 자동 매칭

### Part List 엑셀 매핑 구조

Part List 엑셀(`W5XGVU-SN2709-PLRA`)에서 TAG → 품목코드 매핑:

| 시트 | TAG 컬럼 | 코드 컬럼 | 매핑 수 | 예시 |
|------|---------|----------|---------|------|
| **VALVE LIST** | B열 (CODE) | Y열 (ERP CODE) | 16개 | V01 → `FB2F29-R64A-A111-36L-EXT-GVU-DNV` |
| **SENSOR LIST** | B열 (Instrument Code) | M열 (품목코드) | 22개 | PT01 → `PT-MBS4201-4308G-DANFOSS` |
