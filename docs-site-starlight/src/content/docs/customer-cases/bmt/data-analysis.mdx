## 용어 안내

| 용어 | 의미 |
|------|------|
| **SN (Serial Number)** | 호선번호 — 조선소에서 건조하는 선박 1척의 고유 ID. SN2709와 SN2693은 서로 다른 배 |
| **GVU (Gas Valve Unit)** | 밸브 여러 개 + 센서 + 배관을 하나의 프레임에 조립한 시스템 (= 스키드) |
| **GAR (General Arrangement)** | 스키드 전체 배치도. 밸브 위치(V01~V15)를 보여줌 |
| **VD (Valve Drawing)** | 개별 밸브의 단면도 + 부품리스트(Description 테이블) |
| **ITDRA (Instrument/Tubing Drawing)** | 계장기기(온도센서, 압력센서 등) 스펙 도면 |
| **ERP BOM** | ERP 시스템에 등록된 공식 부품 목록 (자재 발주 기준) |
| **Part List** | 설계 부서에서 관리하는 부품 목록 (도면 PDF 안에 포함되거나 별도 엑셀) |
| **TAG** | 스키드 내 장비 위치 식별자 (예: V01 = 1번 밸브, PT01 = 1번 압력센서) |
| **Balloon 번호** | 도면 이미지에 원형 말꼬리표로 표시된 부품 번호 |

## 1번 자료: 개별 밸브 도면

단품 밸브 FB2FB6의 조립도면입니다. 좌측에 기계 단면도, 우측에 부품리스트 테이블이 있습니다.

![FB2FB6 도면 전체](/images/bmt/fb2fb6-full.png)

- **형식**: Autodesk Inventor .idw → PDF 출력 (벡터, 텍스트 레이어 존재)
- **부품**: 33개 항목번호 (실제 고유 부품 27개)
- **추출 방법**: `pdfplumber` 테이블 자동 감지
- **특이사항**: 항목 24~28이 셀 병합으로 묶여 있어 후처리 필요

## 2번 자료: 스키드(GVU) 도면 내부 구조

22페이지 PDF에 3개 섹션이 합본되어 있습니다.

### GAR 표지 (Page 1) — SN2709

![GAR 표지 - SN2709](/images/bmt/gvu-cover-sn2709.png)

**Doc No.: W5XGVU-SN2709-GAR** — 정상 시리얼 번호

### GAR 배치도 (Page 3)

스키드 전체 배치도로, 밸브 TAG 번호(V01~V15)와 계장기기가 표시됩니다.

![GAR 배치도](/images/bmt/gar-layout.png)

### VD 표지 (Page 4) — SN2693 (불일치!)

![VD 표지 - SN2693](/images/bmt/vd-cover-sn2693.png)

<Aside type="danger" title="CRITICAL: VD 도면 시리얼 불일치">
밸브 상세도(VD) 표지의 DOC 번호가 **W5XGVU-SN2693-VD**입니다.
나머지 전체 자료(GAR, ITDRA, Part List, ERP BOM)는 모두 **SN2709**입니다.
다른 호선의 도면이 잘못 포함된 것으로 확인. SN2709용 VD 도면 재요청 완료 (2026-03-24).

**단, 이 불일치로 막힌 것은 "스키드 내 개별 밸브 하위 부품 비교" 1가지뿐**입니다.
스키드 1단계(TAG→Part List→ERP)와 ITDRA 비교는 모두 SN2709로 정상 테스트 가능하며,
하위 부품 비교 로직 자체는 1번 자료(FB2FB6)에서 이미 검증 완료되었습니다.
VD 재수령은 파이프라인 증명이 아니라 **SN2709의 실제 불일치 검출 결과**를 얻기 위해 필요합니다.
</Aside>

## 3번 그룹 = 2번과 동일 파일

MD5 해시 검증으로 3번 그룹 파일이 2번과 완전 동일함을 확인했습니다.

```
MD5 해시 비교:
2_W5XGVU-SN2709-GARRB.pdf       == 3_3D도면예_1)...GARRB.pdf       ✅ 동일
2_W5XGVU-SN2709-PLRA_Part List  == 3_3D도면예_2)...Part List.xlsx  ✅ 동일
2_W5XHEGVU-SN2709-DNV.xlsx      == 3_3D도면예_3)...DNV.xlsx        ✅ 동일
```

3번 그룹에서 유일하게 다른 파일은 `GVU_3D.pdf` (3D 렌더링 참조 이미지) 1개뿐입니다.
