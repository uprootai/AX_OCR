## 고객 회신 상태 (1차 질문)

| # | 질문 | 상태 |
|---|------|------|
| 1 | 자료 구분 (FB2FB6 = 프로세스밸브, GVU = 스키드) | ✅ 확인 완료 |
| 2 | 도면 3가지 유형 모두 검증 대상 + 번호 없는 도면 존재 | ✅ 확인 완료 |
| 3 | 검증 범위: 파트리스트 수준 + 하위 부품까지 (둘 다) | ✅ 확인 완료 |
| 4 | 휴먼 에러 유형 별도 자료 | 🟡 추후 수령 |
| 5 | 매칭 방식: 스키드 3단계, 개별 밸브 3단계 (보정) | ✅ 확인 완료 |

## 추가 확인 필요 사항

| # | 질문 | 배경 |
|---|------|------|
| 6 | 도면 PDF가 항상 CAD 출력(벡터)인지, 스캔본도 있는지 | 스캔본이면 OCR 필요 |
| 7 | ERP BOM 엑셀 포맷이 항상 동일한지 | 컬럼 순서, 계층 표기 방식 |
| 8 | 피팅&계장밸브 샘플 추가 요청 | 3개 구분 중 미수령 |
| 9 | GA/Layout 도면 샘플 필요 | 부품번호 없는 유형 |
| 10 | Part List가 별도 파일인지, 도면 PDF 내 포함인지 | 3자 비교 필요 여부 |
| 11 | SN2709용 VD 밸브 상세도면 재전달 | **요청 완료** |
| 12 | Part List 내 SN2711 참조 — 다른 호선인지? | W5XGVU-SN2711-PLRA 발견 |
| 13 | 도면에서 안 보이는 부품 → 테이블 기준 검증 맞는지 | 어셈블리 결합 상태 |
