## Test 3: BOM 매칭 (S04)

| 테스트 | 검증 내용 |
|--------|---------|
| `test_part_list_files_exist` | Part List / ERP BOM 엑셀 존재 |
| `test_valve_list_has_tags` | VALVE LIST에 12개+ TAG 매핑 |
| `test_sensor_list_has_tags` | SENSOR LIST에 10개+ TAG 매핑 |
| `test_erp_bom_has_items` | ERP BOM에 50개+ .1 레벨 항목 |
| `test_v01_exact_match` | V01 → `FB2F29-R64A-A111-36L-EXT-GVU-DNV` exact match |
| `test_v06_mismatch_detected` | V06 불일치 검출 (Part List에 `-DNV` 누락) |
| `test_pi02_mismatch_detected` | PI02 불일치 검출 (MAX331 vs MAX311 오타) |
| `test_matching_count` | 전체 매칭 19건+ 확인 |
| `test_total_mismatches` | V06, PI02 불일치 자동 검출 확인 |

<Aside type="tip" title="불일치 검출 테스트">
`test_v06_mismatch_detected`와 `test_pi02_mismatch_detected`는 **의도적으로 매칭 실패를 검증**합니다.
Part List 코드가 ERP BOM에서 exact match로 찾아지면 오히려 테스트 실패 — 불일치가 정확히 검출되는지 확인합니다.
</Aside>
