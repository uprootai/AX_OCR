# 빠른 시작 가이드 (Quick Start Guide)

> 도면 업로드부터 BOM 다운로드까지 6단계로 완료하는 현장 엔지니어용 안내서다.

---

## 시작 전 확인사항

| 항목 | 요건 |
|------|------|
| 브라우저 | Chrome 또는 Edge (최신 버전) |
| 파일 형식 | PDF, PNG, JPG, DWG |
| 파일 크기 | 최대 50MB |
| 네트워크 | 사내 네트워크 연결 필요 |

---

## Step 1 — 로그인 / 프로젝트 선택

브라우저에서 시스템 주소(`http://서버IP:3000`)를 열고 로그인한다.

![로그인 화면](/screenshots/user-guide/01-login.png)

로그인 후 **프로젝트 목록**에서 작업할 프로젝트를 선택한다.

- 새 프로젝트: 우측 상단 **+ 새 프로젝트** 버튼 클릭
- 기존 프로젝트: 카드 클릭으로 바로 진입

![프로젝트 선택](/screenshots/user-guide/01-project-select.png)

:::tip
프로젝트가 보이지 않으면 관리자에게 권한 부여를 요청한다.
:::

---

## Step 2 — 도면 업로드 (Drag & Drop)

프로젝트 대시보드 좌측의 **도면 업로드** 영역으로 이동한다.

![업로드 영역](/screenshots/user-guide/02-upload-zone.png)

파일을 업로드하는 방법은 두 가지다:

1. **드래그 앤 드롭**: 파일 탐색기에서 파일을 끌어다 점선 영역에 놓는다
2. **파일 선택**: **파일 선택** 버튼을 클릭하여 파일 탐색기에서 선택한다

여러 파일을 한 번에 업로드할 수 있다. 업로드 완료 후 썸네일이 표시된다.

![업로드 완료](/screenshots/user-guide/02-upload-complete.png)

:::caution
DWG 파일은 자동으로 PDF로 변환된다. 변환에 30초~2분 소요될 수 있다.
:::

---

## Step 3 — 분석 실행 (워크플로우 선택)

업로드한 도면을 선택한 뒤 **분석 시작** 버튼을 클릭한다.

![분석 시작 버튼](/screenshots/user-guide/03-analyze-button.png)

**워크플로우 선택** 창이 열린다. 용도에 맞는 워크플로우를 선택한다:

| 워크플로우 | 용도 |
|-----------|------|
| 표준 분석 | 일반 기계 도면 |
| 베어링 전용 | 베어링 도면 특화 분석 |
| P&ID 분석 | 배관 계장 도면 |
| 고속 분석 | 빠른 결과 (정확도 낮음) |

![워크플로우 선택](/screenshots/user-guide/03-workflow-select.png)

**분석 실행** 버튼을 클릭하면 분석이 시작된다. 진행률이 화면에 표시된다.

---

## Step 4 — 검출 결과 확인 (바운딩 박스, 치수)

분석이 완료되면 결과 화면으로 자동 이동한다.

![결과 화면](/screenshots/user-guide/04-result-overview.png)

도면 위에 **바운딩 박스**가 표시된다. 색상별 의미:

| 색상 | 의미 |
|------|------|
| 초록 | 신뢰도 높음 (80% 이상) |
| 노랑 | 신뢰도 보통 (50~80%) |
| 빨강 | 신뢰도 낮음 (50% 미만) — 확인 필요 |

바운딩 박스를 클릭하면 우측 패널에 상세 정보가 표시된다:

- 부품 유형 및 식별 코드
- 치수 정보 (OCR 추출 값)
- 신뢰도 점수

![바운딩 박스 상세](/screenshots/user-guide/04-bbox-detail.png)

---

## Step 5 — 결과 검증 (승인/거부/수정)

검출 결과를 검토하고 각 항목을 승인, 거부 또는 수정한다.

![검증 패널](/screenshots/user-guide/05-validation-panel.png)

| 동작 | 버튼 | 단축키 |
|------|------|--------|
| 승인 | ✓ 초록 버튼 | `A` |
| 거부 | ✗ 빨강 버튼 | `R` |
| 수정 | ✎ 수정 버튼 | `E` |
| 다음 항목 | → 버튼 | `Space` |

**수정** 선택 시 직접 값을 편집할 수 있다:

1. 부품명 수정: 드롭다운에서 선택 또는 직접 입력
2. 치수 수정: 숫자 입력란에 올바른 값 입력
3. **저장** 버튼 클릭

![수정 화면](/screenshots/user-guide/05-edit-result.png)

모든 항목 검토 후 **검증 완료** 버튼을 클릭한다.

---

## Step 6 — BOM 다운로드 (Excel/PDF)

검증 완료 후 **BOM 생성** 탭으로 이동한다.

![BOM 탭](/screenshots/user-guide/06-bom-tab.png)

BOM 내용을 확인하고 다운로드 형식을 선택한다:

| 형식 | 용도 |
|------|------|
| Excel (.xlsx) | 추가 편집, ERP 시스템 연동 |
| PDF | 출력, 공유, 문서 보관 |
| CSV | 데이터 분석, 커스텀 가공 |

**다운로드** 버튼을 클릭하면 파일이 즉시 저장된다.

![다운로드 버튼](/screenshots/user-guide/06-download.png)

:::tip
견적서도 함께 생성하려면 **견적서 생성** 버튼을 클릭한다. BOM 기반으로 자동 생성된다.
:::

---

## 완료

6단계가 끝났다. 생성된 BOM 파일을 ERP 또는 구매 시스템에 업로드하여 활용한다.

문제가 발생하면 [FAQ](./faq)를 먼저 확인하거나, 시스템 관리자에게 문의한다.
