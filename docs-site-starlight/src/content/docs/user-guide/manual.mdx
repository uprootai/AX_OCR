# 사용자 매뉴얼 (User Manual)

> AX POC의 모든 기능을 화면별로 설명하는 현장 엔지니어용 참조 매뉴얼이다.

---

## 대시보드 화면 설명

로그인 후 첫 화면이다. 네 개 영역으로 구성된다.

![대시보드 전체](/screenshots/user-guide/manual-dashboard.png)

| 영역 | 위치 | 설명 |
|------|------|------|
| 상단 헤더 | 최상단 | 프로젝트 선택, 알림, 계정 메뉴 |
| 좌측 사이드바 | 좌측 | 메뉴 네비게이션 |
| 중앙 콘텐츠 | 중앙 | 현재 선택된 기능 화면 |
| 우측 패널 | 우측 | 상세 정보, 작업 이력 |

### 대시보드 주요 지표

| 지표 | 설명 |
|------|------|
| 분석 대기 | 처리 대기 중인 도면 수 |
| 오늘 완료 | 오늘 분석 완료 건수 |
| 검증 필요 | 사용자 검토가 필요한 항목 수 |
| 월간 처리량 | 이번 달 누적 처리 도면 수 |

## 워크플로우 빌더 사용법

워크플로우는 도면 분석 처리 순서를 정의한다.

### 기본 제공 워크플로우

| 워크플로우 이름 | 처리 단계 | 권장 대상 |
|----------------|-----------|-----------|
| 표준 분석 | 전처리 → YOLO 검출 → OCR → BOM | 일반 기계 도면 |
| 베어링 전용 | 전처리 → YOLO(베어링) → 치수 추출 | 베어링 어셈블리 도면 |
| P&ID 분석 | 전처리 → 기호 인식 → 태그 추출 | 배관 계장 도면 |
| 고속 분석 | YOLO 검출만 | 빠른 프리뷰 |

### 커스텀 워크플로우 만들기

좌측 메뉴 **워크플로우** → **+ 새 워크플로우** 클릭.

![워크플로우 빌더](/screenshots/user-guide/manual-workflow-builder.png)

1. **이름 입력**: 워크플로우 이름을 입력한다
2. **노드 추가**: 좌측 노드 패널에서 원하는 처리 단계를 캔버스로 드래그한다
3. **연결**: 노드 사이를 마우스로 이어 처리 순서를 정한다
4. **설정**: 각 노드를 클릭하면 세부 파라미터를 조정할 수 있다
5. **저장**: 우측 상단 **저장** 버튼을 클릭한다

| 노드 유형 | 기능 |
|----------|------|
| 전처리 | 이미지 보정, 노이즈 제거 |
| YOLO 검출 | 부품 위치 검출 |
| OCR | 텍스트·치수 인식 |
| BOM 생성 | 검출 결과를 BOM으로 변환 |
| 검증 요청 | 사용자 검토 단계 삽입 |

## 검출 결과 해석

분석 완료 후 결과 화면에서 각 요소를 확인한다.

### Confidence Score (신뢰도 점수)

신뢰도 점수는 AI 모델이 예측한 정확성의 척도다.

| 점수 범위 | 색상 | 처리 방법 |
|----------|------|-----------|
| 80% 이상 | 초록 | 자동 승인 가능 |
| 50~79% | 노랑 | 육안 확인 후 승인 |
| 50% 미만 | 빨강 | 반드시 수동 검토 |

:::caution
빨간색 항목은 반드시 확인해야 한다. 승인하지 않으면 BOM에 포함되지 않는다.
:::

### 바운딩 박스 색상

검출된 영역은 바운딩 박스로 표시된다.

| 색상 | 상태 |
|------|------|
| 초록 테두리 | 승인됨 |
| 노랑 테두리 | 검토 대기 |
| 빨강 테두리 | 신뢰도 낮음, 검토 필요 |
| 회색 테두리 | 거부됨 |
| 파랑 테두리 | 현재 선택됨 |

### 상세 정보 패널

바운딩 박스를 클릭하면 우측 패널에 표시된다:

| 필드 | 설명 |
|------|------|
| 부품 유형 | 검출된 부품 카테고리 |
| 부품 코드 | 식별 코드 (KS 표준 기반) |
| 치수 | OCR로 추출한 수치 |
| 단위 | mm, inch 등 |
| 신뢰도 | 0~100% |
| 수량 | 동일 부품 검출 수 |

## BOM 생성 및 편집

### BOM 자동 생성

검증 완료 후 **BOM 생성** 버튼을 클릭하면 승인된 항목만 모아 BOM이 자동 생성된다.

![BOM 화면](/screenshots/user-guide/manual-bom.png)

### BOM 편집

생성된 BOM은 표 형태로 표시된다. 셀을 직접 클릭하여 편집한다.

| 편집 가능 항목 | 방법 |
|---------------|------|
| 수량 변경 | 수량 셀 클릭 후 숫자 입력 |
| 부품명 수정 | 부품명 셀 더블클릭 후 입력 |
| 행 추가 | 최하단 **+ 행 추가** 클릭 |
| 행 삭제 | 행 좌측 체크박스 선택 후 삭제 |
| 정렬 | 컬럼 헤더 클릭 |

### BOM 열 구성

| 열 | 설명 |
|----|------|
| 번호 | 순번 |
| 부품 코드 | 식별 코드 |
| 부품명 | 한국어 명칭 |
| 규격 | 치수 및 사양 |
| 수량 | 개수 |
| 단위 | EA, SET 등 |
| 비고 | 자유 입력 |

### BOM 다운로드

우측 상단 **다운로드** 버튼에서 형식을 선택한다:

| 형식 | 특징 |
|------|------|
| Excel (.xlsx) | 편집 가능, ERP 연동 |
| PDF | 출력·보관용 |
| CSV | 시스템 연동, 데이터 가공 |

## 견적서 생성/내보내기

### 견적서 생성

BOM 화면에서 **견적서 생성** 버튼을 클릭한다.

![견적서 생성](/screenshots/user-guide/manual-quote.png)

단가 입력 방법:

| 방법 | 설명 |
|------|------|
| 직접 입력 | 각 항목 단가란에 직접 입력 |
| 단가표 연동 | 등록된 단가표에서 자동 불러오기 |
| 공급업체 조회 | 연동 공급업체 단가 자동 적용 |

### 견적서 구성 항목

| 항목 | 내용 |
|------|------|
| 발행일 | 자동 입력 (수정 가능) |
| 유효기간 | 기본 30일 (수정 가능) |
| 부품 목록 | BOM에서 자동 가져옴 |
| 소계/세금/합계 | 자동 계산 |
| 비고 | 자유 입력 |

### 견적서 내보내기

| 형식 | 용도 |
|------|------|
| PDF | 고객사 제출 |
| Excel | 내부 검토·수정 |
| 이메일 전송 | 수신자 이메일 입력 후 바로 전송 |

## 시스템 설정

좌측 메뉴 하단 **설정** 아이콘을 클릭한다.

### API 키 설정

외부 시스템 연동에 사용하는 API 키를 관리한다.

| 항목 | 설명 |
|------|------|
| API 키 발급 | **+ 새 키 발급** 버튼 클릭 |
| 키 복사 | 눈 모양 아이콘으로 확인 후 복사 |
| 키 만료 | 만료일 설정 가능 |
| 키 삭제 | 삭제 시 해당 키 즉시 무효화 |

:::caution
API 키는 발급 시 한 번만 전체 표시된다. 즉시 안전한 곳에 저장한다.
:::

### AI 모델 선택

분석에 사용할 AI 모델을 선택한다.

| 모델 | 특징 | 권장 용도 |
|------|------|-----------|
| YOLO v11 (기본) | 빠름, 범용 | 일반 도면 |
| YOLO v11 Fine-tuned | 베어링 특화 | 베어링 도면 |
| Ensemble | 정확도 최고, 느림 | 고정밀 분석 |

### 알림 설정

| 항목 | 설명 |
|------|------|
| 분석 완료 알림 | 분석 완료 시 브라우저 알림 |
| 이메일 알림 | 분석 완료 시 이메일 발송 |
| 검증 요청 알림 | 검토 필요 항목 발생 시 알림 |
