# 자주 묻는 질문 (FAQ)

> 현장에서 자주 발생하는 질문 10가지와 해결 방법을 정리했다.

---

## Q1. 어떤 파일 형식을 지원하나요?

**지원 형식:**

| 형식 | 확장자 | 비고 |
|------|--------|------|
| PDF | `.pdf` | 권장 형식 |
| 이미지 | `.png`, `.jpg`, `.jpeg` | 300 DPI 이상 권장 |
| CAD 도면 | `.dwg`, `.dxf` | 자동으로 PDF 변환 후 처리 |
| TIFF | `.tif`, `.tiff` | 고해상도 스캔 도면용 |

**파일 크기 제한:** 파일당 최대 50MB

**해상도 권장:** 이미지 파일은 300 DPI 이상이어야 OCR 정확도가 높다. 저해상도 스캔 도면은 신뢰도가 낮게 나올 수 있다.

---

## Q2. 도면 분석에 얼마나 걸리나요?

분석 시간은 도면 복잡도와 선택한 워크플로우에 따라 다르다.

| 워크플로우 | 평균 처리 시간 | 비고 |
|-----------|--------------|------|
| 고속 분석 | 10~30초 | YOLO 검출만 수행 |
| 표준 분석 | 1~3분 | 검출 + OCR |
| 베어링 전용 | 1~2분 | 베어링 특화 모델 |
| Ensemble | 3~8분 | 최고 정확도, 가장 느림 |

**처리 중 다른 작업 가능:** 분석이 시작되면 다른 도면 업로드나 BOM 편집 등을 동시에 진행할 수 있다. 분석 완료 시 알림이 표시된다.

---

## Q3. 검출 정확도가 낮으면 어떻게 하나요?

정확도가 낮을 때 시도할 수 있는 방법:

**1단계 — 이미지 품질 확인**
- 스캔 해상도를 300 DPI 이상으로 높인다
- 도면이 기울어진 경우 똑바로 스캔한다
- 흐리거나 오염된 도면은 원본을 재스캔한다

**2단계 — 워크플로우 변경**
- 도면 유형에 맞는 워크플로우를 선택한다 (베어링 도면 → 베어링 전용)
- **Ensemble** 워크플로우로 재분석하면 정확도가 향상된다

**3단계 — 수동 검증**
- 신뢰도 50% 미만 항목은 직접 확인하고 수정한다
- 바운딩 박스가 잘못된 경우 삭제하고 수동으로 추가할 수 있다

**4단계 — 관리자 문의**
- 특정 도면 유형에서 반복적으로 정확도가 낮으면 관리자에게 보고한다. Fine-tuning을 통해 개선할 수 있다.

---

## Q4. 여러 도면을 한번에 분석할 수 있나요?

가능하다. **배치 분석** 기능을 사용한다.

**방법:**

1. 업로드 영역에서 여러 파일을 한 번에 선택하거나 드래그한다
2. 업로드된 파일 목록에서 분석할 파일에 체크한다
3. **선택 항목 분석** 버튼을 클릭하고 워크플로우를 선택한다
4. 모든 파일이 순차적으로 처리된다

**배치 처리 제한:**

| 항목 | 제한 |
|------|------|
| 최대 파일 수 | 50개 |
| 총 용량 | 500MB |
| 동시 처리 | 최대 5개 병렬 처리 |

각 파일의 진행 상태는 목록에서 실시간으로 확인할 수 있다.

---

## Q5. BOM 데이터를 편집할 수 있나요?

가능하다. BOM은 분석 완료 후 자유롭게 편집할 수 있다.

**편집 가능한 항목:**

| 항목 | 편집 방법 |
|------|-----------|
| 부품명 | 셀 더블클릭 후 직접 입력 |
| 수량 | 셀 클릭 후 숫자 입력 |
| 규격 | 셀 더블클릭 후 수정 |
| 단위 | 드롭다운 선택 |
| 비고 | 셀 클릭 후 자유 입력 |
| 행 추가/삭제 | 우클릭 컨텍스트 메뉴 사용 |

**주의:** BOM을 편집해도 원본 도면과 검출 결과는 변경되지 않는다. 재분석하면 BOM이 초기화되므로, 중요한 편집 내용은 먼저 다운로드해 두는 것이 좋다.

---

## Q6. 견적서 양식을 변경할 수 있나요?

기본 제공 양식을 수정하거나 커스텀 양식을 추가할 수 있다.

**기본 양식 수정 (관리자 권한 필요):**

1. 설정 → **견적서 양식** 탭으로 이동한다
2. 회사 로고, 회사명, 주소, 연락처를 입력한다
3. 컬럼 표시 여부와 순서를 조정한다
4. **저장** 클릭

**커스텀 템플릿 (Excel 기반):**

1. 설정 → **견적서 양식** → **템플릿 업로드**를 클릭한다
2. 준비된 Excel 템플릿(`.xlsx`) 파일을 업로드한다
3. 데이터 매핑 설정 후 저장한다

템플릿에 사용할 수 있는 변수는 **템플릿 가이드** 버튼을 클릭해 확인한다.

## Q7. 오프라인에서도 사용 가능한가요?

현재 버전은 **오프라인 사용을 지원하지 않는다.**

AX POC는 서버 기반 AI 분석 엔진을 사용하므로 서버와의 네트워크 연결이 필요하다.

**네트워크 요건:**

| 항목 | 요건 |
|------|------|
| 최소 속도 | 10 Mbps |
| 권장 속도 | 100 Mbps 이상 |
| 네트워크 유형 | 사내 네트워크 또는 VPN |
| 인터넷 연결 | 불필요 (사내 서버 사용) |

현장 사무소에서 사내 네트워크에 접속이 어려운 경우 VPN 연결 후 사용한다. VPN 설정은 IT 담당자에게 문의한다.

---

## Q8. 데이터는 어디에 저장되나요?

모든 데이터는 **사내 서버에만 저장된다.** 외부 클라우드로 전송되지 않는다.

**저장 위치:**

| 데이터 유형 | 저장 위치 |
|------------|-----------|
| 업로드 도면 | 사내 서버 `/data/uploads/` |
| 분석 결과 | 사내 서버 데이터베이스 |
| BOM/견적서 | 사내 서버 데이터베이스 |
| 사용자 계정 | 사내 서버 데이터베이스 |

**보관 기간:**

| 항목 | 기본 보관 기간 |
|------|--------------|
| 업로드 파일 | 90일 |
| 분석 결과 | 1년 |
| BOM 데이터 | 무기한 (삭제 전까지) |

보관 기간은 관리자 설정에서 변경 가능하다. 데이터 보안 정책은 IT 보안 담당자에게 문의한다.

## Q9. 분석 결과를 공유할 수 있나요?

세 가지 방법으로 결과를 공유할 수 있다.

**방법 1 — 링크 공유:**

결과 화면 우측 상단 **공유** 버튼을 클릭한다. 생성된 링크를 복사하여 전달한다.
- 링크 유효기간: 기본 7일 (설정 변경 가능)
- 링크 수신자는 계정 없이 결과 조회 가능 (읽기 전용)

**방법 2 — 파일 다운로드 후 전달:**

BOM 또는 견적서를 PDF/Excel로 다운로드하여 이메일이나 메신저로 전달한다.

**방법 3 — 이메일 직접 전송:**

견적서 화면에서 **이메일로 보내기** 버튼을 클릭한다. 수신자 이메일을 입력하면 PDF가 첨부된 이메일이 발송된다.

## Q10. 시스템 요구사항은 무엇인가요?

**클라이언트 (사용자 PC):**

| 항목 | 요건 |
|------|------|
| 브라우저 | Chrome 120 이상, Edge 120 이상 |
| OS | Windows 10/11, macOS 13 이상, Ubuntu 20.04 이상 |
| 화면 해상도 | 1920×1080 이상 권장 |
| RAM | 4GB 이상 (8GB 권장) |
| 특별 설치 | 불필요 (브라우저만 있으면 됨) |

**서버 (IT 담당자 참조):**

| 항목 | 최소 사양 | 권장 사양 |
|------|-----------|-----------|
| CPU | 8코어 | 16코어 이상 |
| RAM | 32GB | 64GB |
| GPU | NVIDIA RTX 3080 | A100 또는 동급 |
| 저장소 | 500GB SSD | 2TB NVMe SSD |
| OS | Ubuntu 22.04 LTS | Ubuntu 22.04 LTS |

서버 설치 및 설정은 [배포 가이드](/deployment/)를 참조하거나 시스템 관리자에게 문의한다.

---

## 해결되지 않은 문제가 있나요?

위 항목으로 해결되지 않은 경우:

1. 시스템 관리자에게 문의한다
2. 화면 캡처와 함께 오류 메시지를 전달한다
3. **설정 → 로그 내보내기**로 로그 파일을 첨부하면 빠른 해결에 도움이 된다
