# 라우트 맵 (Route Map)

> web-ui(포트 5173)의 12개 라우트 선언 기준 마스터 레퍼런스이다. 모든 경로는 `App.tsx`의 `<Routes>` 기준이다.

## 라우트 인벤토리

| 구분 | 수 | 설명 |
|------|-----|------|
| 사용자 페이지 | 9개 | 공개 접근 가능 |
| 관리자 페이지 | 2개 | 기능적 관리 목적 (현재 공개) |
| 시스템 라우트 | 1개 | 리다이렉트 |

## 사용자 페이지

인증 없이 접근 가능하다.

| 경로 | 권한 | 페이지 | 컴포넌트 | 로딩 | 설명 | 가이드 |
|------|------|--------|----------|------|------|--------|
| `/dashboard` | 공개 | 대시보드 | `Dashboard` | lazy | 시스템 현황 + 프로젝트 요약 | [상세](./pages/dashboard) |
| `/guide` | 공개 | 가이드 | `Guide` | lazy | 시스템 사용 가이드 | — |
| `/docs` | 공개 | 문서 | `Docs` | lazy | docs-site 브릿지 | — |
| `/blueprintflow/builder` | 공개 | 빌더 | `BlueprintFlowBuilder` | lazy | 워크플로우 캔버스 | [상세](./pages/blueprintflow) |
| `/blueprintflow/list` | 공개 | 목록 | `BlueprintFlowList` | lazy | 워크플로우 CRUD | [상세](./pages/blueprintflow) |
| `/blueprintflow/templates` | 공개 | 템플릿 | `BlueprintFlowTemplates` | lazy | 템플릿 갤러리 | [상세](./pages/blueprintflow) |
| `/pid-overlay` | 공개 | P&ID | `PIDOverlayPage` | lazy | 심볼 검출 오버레이 | [상세](./pages/session) |
| `/projects` | 공개 | 목록 | `ProjectList` | lazy | 프로젝트 목록 | [상세](./pages/project) |
| `/projects/:projectId` | 공개 | 상세 | `ProjectDetail` | lazy | 프로젝트 상세 | [상세](./pages/project) |

## 관리자 페이지

기능적으로 관리 목적이나 현재 권한 분리는 없다.

| 경로 | 권한 | 페이지 | 컴포넌트 | 로딩 | 설명 | 가이드 |
|------|------|--------|----------|------|------|--------|
| `/admin` | 공개* | 시스템 관리 | `Admin` | lazy | 상태, 모델, 로그, 백업 | [상세](./pages/admin) |
| `/admin/api/:apiId` | 공개* | API 상세 | `APIDetail` | lazy | 개별 API 설정 | [상세](./pages/admin) |

\*향후 관리자 인증 추가 예정

## 시스템 라우트

| 경로 | 유형 | 설명 |
|------|------|------|
| `/` | redirect | `/dashboard`로 자동 리다이렉트 |

## 레이아웃 구조

모든 렌더 라우트는 `<Layout />` 내부에서 렌더링된다.

```
<BrowserRouter>
  <Suspense fallback={<PageLoader />}>
    <Routes>
      /                → Navigate to="/dashboard"
      <Layout>         (11개 라우트)
        /dashboard
        /guide
        /docs
        /blueprintflow/builder
        /blueprintflow/list
        /blueprintflow/templates
        /admin
        /admin/api/:apiId
        /pid-overlay
        /projects
        /projects/:projectId
      </Layout>
    </Routes>
  </Suspense>
</BrowserRouter>
```

## 로딩 전략

| 항목 | 값 |
|------|-----|
| 동기 로드 | `Layout` (1개) |
| 지연 로드 | 11개 페이지 (React.lazy) |
| 폴백 | `<PageLoader />` 스피너 |
| React Query | staleTime 30초, refetchOnWindowFocus 비활성 |

## 관련 API

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| GET | `Gateway /health` | 전체 서비스 헬스체크 |
| GET | `Gateway /admin/status` | 시스템 상태 (API + GPU + 리소스) |
