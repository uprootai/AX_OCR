# 시스템 관리 (Admin Page)

> 시스템 현황 모니터링, ML 모델 관리, Docker 로그 조회, 설정 백업/복원, 개별 API 상세 설정을 수행하는 관리 페이지이다.

## 접속 방법

| URL | 권한 | 설명 |
|-----|------|------|
| `/admin` | 공개 | 시스템 관리 메인 (4개 탭) |
| `/admin/api/:apiId` | 공개 | 개별 API 상세 설정 |

<a id="admin-main"></a>

## Admin 메인 (`Admin`)

컴포넌트: `pages/admin/Admin.tsx`

### 탭 구성

| 탭 | 아이콘 | 설명 | 주요 동작 |
|----|--------|------|----------|
| 시스템 현황 | Server | API 상태 카드, GPU 상태, 시스템 리소스(CPU/Memory/Disk) | 5초 자동 갱신 |
| 모델 관리 | Database | ML 모델 경로, 파일 수, 총 용량 표시 | 수동 새로고침 |
| 로그 조회 | FileCode | 서비스 드롭다운 → Docker 로그 표시 (터미널 스타일) | 서비스 선택, 새로고침 |
| 백업/복원 | Archive | 설정 내보내기(JSON), 가져오기, 기본값 초기화 | Export/Import/Reset |

### 시스템 현황 탭 상세

- **API 상태**: 각 API의 display_name, 포트, healthy/unhealthy, GPU 사용 여부, 응답 시간(ms)
- **GPU 상태**: 디바이스명, VRAM 사용량/전체, 사용률(%) 프로그레스 바
- **시스템 리소스**: CPU/Memory/Disk 사용률(%) 프로그레스 바

### 백업/복원 탭 상세

| 동작 | 설명 |
|------|------|
| 설정 내보내기 | localStorage의 serviceConfigs, hyperParameters, api-config-store를 JSON 다운로드 |
| 설정 가져오기 | JSON 파일에서 복원 (버전 확인, 덮어쓰기 경고) |
| 기본값 초기화 | serviceConfigs, hyperParameters 삭제 후 새로고침 |

<a id="api-detail"></a>

## API 상세 (`APIDetail`)

컴포넌트: `pages/admin/APIDetail.tsx`

개별 API 서비스의 설정, Docker 제어, 하이퍼파라미터를 관리한다. URL의 `:apiId` 파라미터로 대상 API를 식별한다.

### 구성 모듈

| 모듈 | 파일 | 설명 |
|------|------|------|
| `useAPIDetail` | `api-detail/hooks/useAPIDetail.ts` | 상태 관리, 데이터 페칭 |
| `ServiceSettingsCard` | `api-detail/components/ServiceSettingsCard.tsx` | 서비스 설정 UI |
| `DockerControlCard` | `api-detail/components/DockerControlCard.tsx` | Docker 시작/중지/재시작/삭제 |
| `APIKeySettingsPanel` | `api-detail/components/APIKeySettingsPanel.tsx` | 외부 API 키 관리 |
| `HyperparamEditor` | `api-detail/components/HyperparamEditor.tsx` | 하이퍼파라미터 편집 |
| `YOLOModelManager` | `components/admin/YOLOModelManager.tsx` | YOLO 모델 전용 관리 |

### 주요 기능

| 기능 | 설명 |
|------|------|
| 서비스 설정 | API별 설정 조회/수정 |
| Docker 제어 | 컨테이너 시작, 중지, 재시작, 삭제 |
| GPU 정보 | GPU 할당 상태 표시 |
| 로그 조회 | 서비스별 Docker 로그 |
| API 키 | 외부 API 키 등록/테스트 |
| 하이퍼파라미터 | 서비스별 파라미터 편집/저장 |
| API 스펙 | OpenAPI 스펙 링크 |

<a id="related-api"></a>

## 관련 API

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| GET | `Gateway /admin/status` | 시스템 상태 (API + GPU + 시스템 리소스) |
| GET | `Gateway /admin/models` | 전체 모델 경로 정보 |
| GET | `Gateway /admin/models/:type` | 특정 모델 타입 정보 |
| GET | `Gateway /admin/logs/:service` | 서비스 Docker 로그 |
| POST | `Gateway /admin/docker/:action/:service` | Docker 컨테이너 제어 (start/stop/restart/remove) |
| GET | `Gateway /admin/docker/ps` | Docker 컨테이너 목록 |
