# 페이지 상세 (Page Details)

> web-ui의 주요 페이지별 구성, 기능, 사용 API를 정리한 문서 목록이다.

## 문서 목록

| 페이지 | 라우트 | 주요 기능 |
|--------|--------|----------|
| [대시보드](./dashboard) | `/dashboard` | API 상태 모니터링, 컨테이너 관리, 프로젝트 현황, 빠른 실행 |
| [프로젝트](./project) | `/projects`, `/projects/:projectId` | 프로젝트 CRUD, 세션 관리, BOM/P&ID 워크플로우 |
| [BlueprintFlow](./blueprintflow) | `/blueprintflow/*` | 워크플로우 빌더, 목록, 템플릿 갤러리 |
| [세션 & P&ID](./session) | `/pid-overlay` | P&ID 심볼 검출 뷰어 |
| [시스템 관리](./admin) | `/admin`, `/admin/api/:apiId` | 시스템 현황, 모델 관리, 로그, 백업, API 상세 |

## 관련 API

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| GET | `Gateway /health` | 전체 서비스 헬스체크 |
| GET | `Gateway /admin/status` | 시스템 상태 조회 |
| GET | `Gateway /projects` | 프로젝트 목록 |
